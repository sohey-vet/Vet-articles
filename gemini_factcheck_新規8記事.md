# 📄 記事1: ショック時の輸液戦略 ─ ボーラス投与の実際

## タグ: 輸液 / 救急 / 2024 AAHA
## 参照論文数: 6本

### 本文の要点:
- ショック時の輸液は「大量急速投与」から「少量ボーラス＋再評価」へパラダイムシフト
- 犬: 等張晶質液 10〜20mL/kg を15〜20分かけて投与 → 再評価 → 必要に応じて追加（最大40〜60mL/kg）
- 猫: 等張晶質液 5〜10mL/kg を15〜20分かけて投与 → 再評価（猫の循環血液量は約55〜60mL/kg）
- 旧来の「犬90mL/kg、猫60mL/kgを一気に投与」は過剰輸液・肺水腫のリスクがあり廃止
- ショックインデックス（SI = 心拍数 ÷ 収縮期血圧）＞1.0 で低灌流を示唆
- 乳酸値＞4mmol/L = 組織低灌流の指標。犬の正常値＜2.5mmol/L
- 輸液反応性 positive: ボーラス後に心拍数↓、血圧↑、乳酸↓、CRT改善
- 輸液反応性 negative: 2回のボーラスで改善なし → 昇圧薬への移行
- 昇圧薬第一選択: ノルエピネフリン 0.1〜2μg/kg/min CRI
- HES（人工膠質液）は急性腎障害リスクのため使用が大きく制限
- 高張食塩水（7.5%）3〜5mL/kg: 頭部外傷合併例で脳圧上昇を避けつつ循環改善
- 猫の未診断HCM（約15〜20%）→ 輸液で肺水腫のリスク
- 日本のチワワ・トイプードル等の小型犬は体重あたりの輸液量が相対的に大きくなり過剰輸液に注意

# 📄 記事2: GDV ─ 安定化と手術判断

## タグ: 救急 / 外科 / GDV
## 参照論文数: 7本

### 本文の要点:
- GDV（胃拡張-捻転症候群）は大型・超大型犬（グレートデン、セントバーナード等）に多く、致死率未治療で100%
- トリアージ: ABCの評価 → 太い静脈路2本（18G以上）
- 初期輸液: ショック用量（等張晶質液 10〜20mL/kg ボーラス × 反復）
- 胃減圧: 経口胃管がゴールドスタンダード → 口から胃管を挿入し、ガスと液体を排出
- 胃管挿入不能時は右パラコスタル部より18G針で経皮穿刺減圧（トロカール法）
- 手術判断: 安定化に6時間以上反応しない、または安定化できた場合でも早期手術推奨（24時間以内）
- 術中: 胃壁の漿膜面の色調（黒変・薄緑色 → 壊死）で切除範囲を決定
- 脾臓の捻転・梗塞があれば脾臓摘出
- 胃固定術（Gastropexy）: 再発予防に必須。インシジョナル法が最も一般的
- 再発率: 胃固定なし → 80%、胃固定あり → 5%以下
- 術後不整脈: 40〜70%で心室性不整脈（VPC/VT）が発生。術後24〜72時間がピーク
- 術後VTにはリドカイン 2mg/kg IVボーラス → CRI 25〜80μg/kg/min
- 術後死亡率: 10〜33%（壊死あり・脾臓摘出あり例では最大42%）
- 日本ではグレートデン等の超大型犬は少ないが、ゴールデン・ラブラドール・秋田犬で発生。柴犬でも報告あり
- 予防的胃固定術: リスク犬種では避妊/去勢手術時に同時実施を推奨する動きあり

# 📄 記事3: ピモベンダンの使い方と最新エビデンス

## タグ: 循環器 / 薬理 / EPIC試験
## 参照論文数: 7本

### 本文の要点:
- ピモベンダン（商品名: ベトメディン®）は「イノダイレーター」（陽性変力作用＋血管拡張作用）
- 作用機序: PDE III阻害 → cAMP↑ + Ca²⁺感受性増強 → 心筋収縮力↑ + 全身・肺血管抵抗↓
- EPIC試験（2016年、Boswood Aら）: 多施設RCT、360頭のMMVDステージB2犬
  - ピモベンダン群: 心不全発症/心臓死までの中央値1228日
  - プラセボ群: 766日
  - ピモベンダン群は心不全発症を約15ヶ月（462日）延長
- ステージB2の判定基準: LA:Ao ≥ 1.6、LVIDdN ≥ 1.7（2019 ACVIM Consensus）
- 用量: 0.25〜0.3mg/kg PO BID（食事の1時間前が望ましい）
- 注射薬（ベトメディン注®）: 急性心不全時に0.15mg/kg IV。強アルカリ性のため血管外漏出注意
- 猫のHCM: ピモベンダンの使用は議論あり。Gordon SGらの研究ではHCM猫50頭に投与し安全性確認（JVIM 2012）
- 猫のHCMでピモベンダン使用する場面: 収縮不全（FS＜25%）が明らかな場合 → 0.625〜1.25mg/猫 PO BID
- DCM: 犬のDCMにはピモベンダン＋標準治療で生存期間延長（PROTECT試験は中断だが傾向データあり）
- 大動脈弁狭窄症には禁忌（閉塞性肥大→圧格差↑）
- ベトメディン®チュアブル錠は日本で最も流通している製剤

# 📄 記事4: 猫の尿道閉塞 ─ 救急対応フロー

## タグ: 救急 / 泌尿器 / 猫
## 参照論文数: 7本

### 本文の要点:
- 猫の尿道閉塞（UO）は雄猫の救急疾患として最頻の一つ。原因は尿道栓子（蛋白基質＋結晶）が最多
- 閉塞12〜24時間で高K血症 → 致死的不整脈（徐脈・心停止）
- 高K血症の緊急治療 → ①グルコン酸Ca 10% 0.5〜1.0mL/kg IV（心筋保護、Kは下げない）→ ②レギュラーインスリン 0.25〜0.5IU/kg IV + 50%デキストロース 1〜2mL/kg IV（細胞内にK移動）
- 心電図所見: K＞6.5mEq/L → T波増高、K＞7.5mEq/L → QRS幅拡大・徐脈、K＞8.5mEq/L → Sine wave → Vfib/心停止
- カテーテル留置: 3.5Fr赤ゴムカテーテルが一般的。閉塞部位での無理な通過は尿道損傷のリスク
- 逆行性フラッシュ: 生理食塩水で尿道を膨らませながら栓子を膀胱内に押し戻す
- 閉塞解除後の利尿: 最初の6〜12時間は閉塞後利尿（post-obstructive diuresis）で大量の希釈尿が生成
- 閉塞後利尿期の輸液: 維持量の2〜3倍（犬猫で4〜6mL/kg/h程度）で脱水を予防
- 閉塞解除翌日から輸液速度を漸減。通常2〜3日で利尿は正常化
- カテーテルは24〜48時間留置後に抜去。抜去後6〜8時間以内に自力排尿を確認
- 再閉塞率: 24〜58%（研究により差がある）
- 再閉塞予防: ウェットフード中心の食事変更、環境エンリッチメント、ストレス軽減
- 繰り返す閉塞には会陰尿道造瘻術（PU手術）を検討。PU後の再閉塞率は約10〜20%
- 日本の一般病院でPU手術は実施可能だが、術者の経験が予後に影響

# 📄 記事5: 維持輸液の考え方 ─ 過剰輸液のリスク

## タグ: 輸液 / 基礎 / 2024 AAHA
## 参照論文数: 6本

### 本文の要点:
- 維持輸液の目的は「飲食できない間の正常な水分・電解質バランスの維持」であり、脱水補正やショック蘇生とは目的が異なる
- 2024 AAHAガイドライン: 維持輸液速度の出発点を犬で約2〜3mL/kg/h、猫で約2mL/kg/hとし、個々の患者に合わせて調整
- 旧来の「1日の維持量＝60mL/kg/日」をルーチンで使うと、特に小型犬・猫・心疾患のある患者で過剰輸液になるリスク
- 輸液の3つの目的を区別: ①蘇生輸液（ボーラス）②脱水補正 ③維持輸液
- 蘇生輸液: 犬10〜20mL/kg/15〜20分、猫5〜10mL/kg/15〜20分のボーラス
- 脱水補正量（mL）= 体重（kg）× 脱水率（%）× 10 → 6〜24時間かけて補正
- 旧来の体表面積法: 30 × BW(kg) + 70（mL/日）→ 2kg以下の超小型犬や猫では過大評価になりやすい
- 過剰輸液の徴候: ①呼吸数の増加（最も早期の指標）②湿性ラ音 ③漿液性鼻汁 ④結膜浮腫（ケモーシス）⑤体重増加＞10% ⑥四肢の浮腫
- 過剰輸液の合併症: 肺水腫、胸水・腹水、消化管浮腫、創傷治癒遅延、希釈性低アルブミン血症
- 24時間で体重が5%以上増加していたら過剰輸液を疑う
- 過剰輸液への対応: ①輸液速度を半減 or 中止 ②フロセミド 1〜2mg/kg IV（重度の場合）③呼吸状態の確認
- 猫の循環血液量: 約55〜60mL/kg（犬の約80〜90mL/kgに比べ少ない）
- 猫の約15〜20%に未診断のHCM → 輸液で肺水腫のリスク
- 心疾患が疑われる猫: 1〜1.5mL/kg/h でさらに慎重に
- 再評価: 4〜6時間ごと（重症患者は2時間ごと）。体重測定は1日2回（朝・夕）
- 尿量目標: ≥1mL/kg/h

# 📄 記事6: 不整脈の救急対応 ─ 心電図の読み方

## タグ: 循環器 / 救急 / 心電図
## 参照論文数: 7本

### 本文の要点:
- 救急不整脈の対応原則: 「不整脈そのものを治すのではなく、血行動態への影響を治療する」
- 心電図の読み方3ステップ: ①心拍数は？ ②QRSの幅は？ ③リズムは規則的か？
- 犬の正常心拍数: 60〜160 bpm（大型犬は低め）、猫: 140〜220 bpm
- P波: 犬 幅＜0.04秒・高さ＜0.4mV、猫 幅＜0.04秒・高さ＜0.2mV
- QRS幅: 犬＜0.06秒（大型犬＜0.05秒）、猫＜0.04秒
- PR間隔: 犬 0.06〜0.13秒、猫 0.05〜0.09秒
- 心室頻拍（VT）: 幅広いQRS（＞0.06秒）が3拍以上連続。P波とQRSの解離
- VT治療適応: 持続性VT（＞30秒）かつ血行動態に影響 → リドカイン投与
- 散発性VPC（単発・二段脈程度）→ 原因検索のみ。治療不要
- リドカイン投与法: 初回ボーラス 2mg/kg IV（2分以上かけて）→ CRI 25〜80μg/kg/min で維持
- 初回不十分 → 2mg/kgを最大3回まで繰り返し可能（総量6mg/kg以内）
- 猫へのリドカイン: 0.25〜0.5mg/kg IVと低用量（猫はリドカインに感受性が高い）
- リドカイン不応VT → ソタロール 1〜2mg/kg PO BID、メキシレチン 5〜8mg/kg PO TID
- 多形性VT（torsades de pointes）→ MgSO4 0.15〜0.3mEq/kg IV 5〜10分
- SVT: QRS幅正常、HR＞250〜300 bpm。迷走神経刺激（眼球圧迫15〜30秒）で診断的＋治療的
- 心房細動（AF）: P波消失、f波、R-R間隔完全に不整。レートコントロールが目標（HR≤140〜160 bpm）
- AF薬物: ジルチアゼム 0.5〜1.5mg/kg PO TID ± ジゴキシン
- 徐脈: 犬HR＜60 bpm、猫HR＜140 bpm
- 房室ブロック分類: 1度（PR延長）、2度Mobitz I型（PR徐々に延長→脱落）、2度Mobitz II型（突然脱落）、3度（完全解離）
- アトロピン反応試験: アトロピン 0.04mg/kg IV → 15〜20分後に再検。反応なし＝器質的伝導障害 → ペースメーカー
- ペースメーカーまでの橋渡し: テルブタリン 0.01mg/kg SC q8h
- 旧来「VPCが出たらすぐリドカイン」→ 現在は散発性VPCは治療不要。「R-on-T」も単独で治療指標にならない
- リドカイン不応時にプロカインアミド → 現在はソタロールが選択肢。プロカインアミドの使用は減少

# 📄 記事7: 蛋白尿の評価と治療介入のタイミング

## タグ: 腎泌尿器 / 糸球体 / IRIS
## 参照論文数: 7本

### 本文の要点:
- 蛋白尿は腎臓病の独立した進行因子であり、早期に検出・介入することで腎機能低下を遅らせる可能性がある
- 尿試験紙は偽陽性が多いため、UPC（尿蛋白/クレアチニン比）での定量が必須
- IRISガイドライン: 犬でUPC＞0.5、猫でUPC＞0.4を持続的蛋白尿と定義し治療介入を推奨
- UPC分類（犬）: ＜0.2=非蛋白尿、0.2〜0.5=境界域、＞0.5=蛋白尿、＞2.0=高度蛋白尿（糸球体疾患を強く示唆）
- UPC分類（猫）: ＜0.2=非蛋白尿、0.2〜0.4=境界域、＞0.4=蛋白尿
- UPC測定は随時尿（スポット尿）で可能。24時間蓄尿は不要
- 1回の測定で判断せず、2〜4週間間隔で2回以上測定して持続性を確認
- 糸球体疾患を疑う所見: UPC＞2.0（犬で＞3.5は高度な糸球体蛋白尿）
- ネフローゼ症候群: 高度蛋白尿 + 低アルブミン血症（＜2.0g/dL）+ 高コレステロール血症 + 浮腫/腹水
- 犬の糸球体疾患の最多原因: 免疫複合体性糸球体腎炎（ICGN）
- 原因検索: 感染症（フィラリア、リーシュマニア等）、内分泌（クッシング）、腫瘍（リンパ腫・多発性骨髄腫）、免疫介在性（SLE・IMHA）、遺伝性（アミロイドーシス）
- 腎生検の適応: UPC＞2.0が持続し原因不明、低アルブミン血症進行
- 第一選択: ACE阻害薬 ベナゼプリル（フォルテコール®）0.5〜1mg/kg PO SID〜BID
- ARB: テルミサルタン（セミントラ®）猫1〜2mg/kg PO SID、犬1mg/kg PO SID → 猫CKDの蛋白尿に日本で承認済み
- ACEi + ARBの併用: 単剤で不十分な場合に追加効果あるが、低血圧・高K・腎機能悪化リスク → 慎重なモニタリング必須
- 治療目標: UPCの50%以上低下、またはUPC＜0.5（犬）/ ＜0.4（猫）
- ACEi開始後にCreが30%以上上昇する場合は減量 or 中止を検討（＜30%は許容範囲）
- 高血圧の合併: 収縮期血圧＞160mmHgは治療対象 → アムロジピン 0.1〜0.25mg/kg PO SID の追加
- 猫のCKDでは高血圧の合併が約20〜60%
- 高度蛋白尿＋低アルブミン例では血栓塞栓症リスク高 → 抗血栓療法（クロピドグレル等）を検討
- 日本ではフィラリアが糸球体疾患の原因として依然重要

# 📄 記事8: 猫の脂肪肝 ─ 強制給餌と栄養管理

## タグ: 肝臓 / 栄養 / 猫
## 参照論文数: 7本

### 本文の要点:
- 猫の肝リピドーシス（HL: Hepatic Lipidosis）は食欲不振が3〜7日以上続くことで発症。特に肥満猫でリスク高
- 猫特有の脂質代謝（VLDL生合成のボトルネック）により、絶食時に肝臓に中性脂肪が蓄積
- 適切な栄養支持を行えば生存率は約80〜85%
- 原発性HLは約5〜15%。大多数は二次性（膵炎・IBD・リンパ腫・糖尿病などが原疾患）
- 栄養回復には4〜8週間を要することが多い
- 典型的プロフィール: 中年〜高齢の室内飼育猫（平均7〜8歳）、肥満〜過体重（BCS ≥7/9）
- 検査所見: ALT/ALP中等度〜高度上昇（ALP＞ALTが多い）、ビリルビン上昇→黄疸、低K/低P/低Mg
- 凝固系: PT/APTT延長 → ビタミンK1投与を検討
- 腹部超音波: 肝臓の高エコー性（腎臓より高輝度）
- 確定診断: 超音波ガイド下の肝臓FNA → 肝細胞内に多量の脂肪空胞（＞50%の肝細胞に脂肪蓄積）
- 凝固系異常がある場合はFNA前にビタミンK1を投与（0.5〜1.5mg/kg SC）
- HLの約85%は二次性 → 膵炎（fPLI）、IBD、リンパ腫、糖尿病、胆管炎を検索
- 経食道チューブ（E-tube）がゴールドスタンダード: 14〜16Frチューブ、短時間全身麻酔で10〜15分で設置
- シリンジによる経口強制給餌はストレス・誤嚥性肺炎リスクが高い → E-tubeが推奨
- E-tube設置部位: 左頸部（食道への経皮的アプローチ）
- 適切なチューブ長: 口から第7〜8肋骨の高さまで
- 在宅管理: 給餌前後に温水5〜10mLでフラッシュ。1回量は10〜15mL/kg程度、1日4〜6回
- RER（安静時エネルギー要求量）= 70 × (体重kg)^0.75 kcal/日
- 簡易式（2〜5kgの猫）: 30 × 体重(kg) + 70 kcal/日
- 段階的増量プロトコル: Day1 = RERの1/3、Day2 = 2/3、Day3〜 = 100%
- 急な全量投与は再給餌症候群（K・P・Mg急降下 → 不整脈・溶血・筋力低下）のリスク
- フードの選択: 高蛋白・高カロリー缶詰（Hill's a/d、ロイヤルカナン退院サポート等）。蛋白含量ME比で30〜35%以上
- サプリメント: ビタミンK1 0.5〜1.5mg/kg SC q12h×3回、ビタミンB群、L-カルニチン 250〜500mg/猫/日、タウリン 250mg/猫/日
- 低P（＜2.5mg/dL）は溶血性貧血の原因 → リン酸Kの静脈内補充
- 嘔吐のコントロール: マロピタント（セレニア®）1mg/kg SC/IV SID が第一選択
- メトクロプラミド 0.2〜0.5mg/kg SC q8h or CRI、オンダンセトロン 0.1〜0.5mg/kg IV q8〜12h
- 電解質管理: 低K → KCl 20〜40mEq/L添加、低P → K2HPO4 0.01〜0.06mmol/kg/h IV、低Mg → MgSO4 0.75〜1mEq/kg/day CRI
- 肝性脳症: ラクツロース 0.5mL/kg PO q8〜12h
- HLの死亡例の多くは再給餌症候群 or 嘔吐コントロール不能
- チューブ抜去の基準: 自力で3日連続食べたらOK（通常4〜8週間）
- 「脂肪肝だから脂肪制限」は誤り → 高蛋白食が基本。十分な蛋白質（30〜35%以上）が必要
