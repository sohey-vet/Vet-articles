## 🧵 Threads用（長文・専門的）

```text
シニア期の健康診断は、単なる習慣ではありません。見た目には元気な動物でも30〜45%から新たな異常が発見されます。特に9歳以上の猫では、慢性腎臓病が15〜30%、甲状腺機能亢進症が10〜15%と高頻度な発症。

腎臓のろ過機能が25〜40%低下した段階で上昇を検出するSDMAは、慢性腎臓病の早期発見に有用なバイオマーカーです。基準値を超える、または尿比重低下を伴う場合、初期の慢性腎臓病として早期介入が重要。病気の進行を遅らせ、生存期間を約2.5倍延長させる重要なエビデンスです。

適正体重（理想的な体型）の維持は、寿命を約1.8〜2.0年延長させる最も確かな予防医療。具体的な寿命延長を伝え、個別の減量プログラム提示も推奨されます。体型が適切でも筋量低下を見落とさないよう、触診での筋量評価を定期化。筋量減少は潜在疾患のサインです。

シニア期には、無症状でも腹部超音波検査の実施とベースラインデータ取得を推奨します。偶発的病変の発見は、後の診断や説明の納得度を高めるため。心臓バイオマーカーNT-proBNPも、高齢動物の心臓病（肥大型心筋症や僧帽弁閉鎖不全症）スクリーニングに有用で、経時トレンドを追う視点も重要です。

最新の血液検査（リキッドバイオプシー）は、血中遊離DNAのゲノム異常検出により、リンパ腫や血管肉腫、骨肉腫などの早期発見に有効なツール。尿検査による膀胱がんなどの診断（尿中BRAF変異検査）も、非侵襲的で高い精度を示す。

これらの早期介入は、QOL向上だけでなく、飼い主様との大切な時間を確実に延長させるもの。健診をルーティンとせず、一歩踏み込んだ提案が今、求められます。
```

その健診、本当に『最適化』されていますか？
臨床獣医師が知るべき
シニア犬猫の健康診断 最適化戦略

[2枚目: 結論]
【臨床獣医師へ】健康診断の最適化 3つの要点
1.  **シニア犬猫の30〜45%**で新規異常を発見。9歳以上の猫ではCKD、甲状腺機能亢進症が高頻度。
2.  **SDMA**はクレアチニンより早期（GFR低下25〜40%）に上昇。感度約90%の必須バイオマーカー。
3.  **適正体重維持（BCS 4〜5/9）**は寿命を約1.8〜2.0年延長する、最もエビデンスレベルの高い予防医療。

[3枚目: なぜ健診が必要か？潜在疾患の発見率]
見かけ上健康な犬猫でも、定期健診で高確率に潜在疾患が発見されます。
| 年齢層 | 異常発見率 |
| :----------------- | :---------- |
| 若齢〜成期（1〜6歳） | 約15〜20% |
| シニア期（7歳以上） | **30〜45%** |

【シニア期に見つかりやすい潜在疾患】
*   猫のCKD（9歳以上）：15〜30%
*   猫の甲状腺機能亢進症：10〜15%
*   犬の脾臓結節・胆泥症等（偶発所見）：25〜30%
*   肥満（BCS 6/9以上）：30〜50%

[4枚目: 年齢別スクリーニングプロトコル：基礎項目]
【全年齢対象 基礎スクリーニング】
1.  **身体検査**
    *   体重測定
    *   **BCS**（ボディコンディションスコア: 1〜9）＋ **MCS**（筋肉コンディションスコア: 正常/減少）を毎回記録
2.  **臨床病理検査**
    *   CBC、血液生化学、尿検査（尿比重・尿タンパク・沈渣）、糞便検査
3.  **感染症スクリーニング**
    *   犬: フィラリア＋ダニ媒介性疾患（4Dxなど）
    *   猫: FeLV/FIVスクリーニング

[5枚目: シニア期追加スクリーニング（7〜8歳以上）]
【シニア期追加スクリーニング（7〜8歳以上）】
1.  **SDMA**
    *   **意義**: クレアチニンより早期（GFR 25〜40%低下）に腎機能低下を検出。感度約90%。
    *   **基準**: >14 µg/dLで精査
2.  **総T4（猫）**
    *   **意義**: 甲状腺機能亢進症のスクリーニング。
    *   **基準**: 基準値上限超で精査
3.  **血圧測定**
    *   **意義**: 高血圧による標的臓器障害の早期発見。
    *   **基準**: 収縮期血圧 >160 mmHgで高血圧

[6枚目: シニア期追加スクリーニング（続き）]
【シニア期追加スクリーニング（7〜8歳以上）】
4.  **NT-proBNP**
    *   **意義**: HCM・MMVDの無症候性スクリーニング。
    *   **特徴**: 感度約85%、特異度約90%。カットオフは犬種・猫で異なる。
5.  **画像診断**
    *   **意義**: 胸部・腹部X線＋腹部超音波検査によるベースライン取得。
    *   偶発所見や潜在疾患の早期発見に大きく寄与。

[7枚目: 早期発見に基づく治療介入]
【早期発見に基づく治療介入】
1.  **CKDの早期介入**
    *   **診断基準**: SDMA >14 µg/dL ＋ 尿比重低下（犬 <1.030、猫 <1.035）→ IRIS Stage 1〜2
    *   **介入**: 早期からの腎臓病用療法食への切り替え、リン吸着剤の導入。
    *   **最新知見**: FGF-23がリン代謝異常のより早期予測マーカーとして注目。
2.  **肥満とサルコペニアの管理**
    *   **肥満（BCS >5/9）**: 目標体重設定、高タンパク・高繊維食による減量プログラム。
    *   **MCS低下**: 基礎疾患精査（腫瘍、CKD、心不全等）、良質なタンパク質供給。

[8枚目: 最新の知見：リキッドバイオプシー]
【最新の知見：リキッドバイオプシー】
1.  **犬のがんスクリーニング（血中cfDNA検査）**
    *   **方法**: NGSを用いた血中遊離DNA（cfDNA）のゲノム異常検出（例: OncoK9）。
    *   **検出対象**: リンパ腫、血管肉腫、骨肉腫等。
    *   **精度**: 特異度 >95%、感度 50〜85%。
    *   ハイリスク犬種に対する健診オプションとして普及中。
2.  **尿中BRAF変異検査（移行上皮癌）**
    *   **意義**: 移行上皮癌の非侵襲的診断。
    *   **精度**: 感度約85%、特異度100%。
    *   血尿を呈する高齢犬・スコティッシュ・テリア等のスクリーニングに有用。

[9枚目: まとめ]
【まとめ】臨床獣医師が実践すべき3つの要点
1.  **シニア期は潜在疾患の宝庫**: 見た目が元気でも30〜45%で新規異常。早期発見が治療成功の鍵。
2.  **SDMAとBCS/MCSは必須**: 腎機能早期マーカーSDMA、適正体重・筋量維持は寿命延長に直結するエビデンスあり。
