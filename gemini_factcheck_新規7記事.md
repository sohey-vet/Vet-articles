# 📄 記事1: 犬猫の中毒 ─ 原因物質と対処法

## タグ: 救急 / 中毒 / 犬猫共通
## 参照論文数: 7本

### 本文の要点:
- チョコレート中毒: テオブロミンの犬LD50は100〜250mg/kg。ミルクチョコ（約1.5mg/g）vsダークチョコ（約5〜8mg/g）vsベーキングチョコ（約14〜16mg/g）
- チョコレート1〜2mg/kg テオブロミンで消化器症状、10mg/kgで心毒性、20mg/kgで痙攣・致死
- キシリトール: 犬は0.1g/kgで低血糖、0.5g/kgで肝不全リスク。猫はキシリトールに比較的耐性
- ブドウ/レーズン: 酒石酸（tartrate）が腎毒性の原因物質として2021年に同定（ASPCA報告）
- ブドウの中毒量は個体差が大きく、ブドウ1粒でもAKIを起こした報告あり
- ユリ: 猫に対して全草が猛毒。花粉を舐めただけでもAKIを起こし得る。犬はユリに対して耐性
- 日本の臨床で多い中毒: チョコレート、玉ねぎ（酸化ヘモグロビン → ハインツ小体性溶血性貧血）、NSAIDs誤飲、ブドウ
- 催吐の適応: 摂取後2時間以内、意識清明、腐食性物質でない
- 催吐薬: 犬 → 3%過酸化水素1〜2mL/kg PO（最大45mL）。日本では入手困難でトラネキサム酸50mg/kg IVが代替
- 活性炭: 1〜2g/kg PO。腐食性物質・炭化水素には禁忌
- ASPCAデータ: 2023年に10万件以上の中毒相談（過去最多）。OTC薬品（17%）、食物（14%）、チョコレート（10%）が上位3

# 📄 記事2: ショック管理 ─ 最新プロトコル

## タグ: 救急 / ショック / 輸液
## 参照論文数: 7本

### 本文の要点:
- ショックの4類型: ①循環血液量減少性 ②心原性 ③分布異常性（敗血症性・アナフィラキシー） ④閉塞性
- 犬のショックボーラス: 10〜20mL/kg を15〜20分で投与 → 再評価 → 追加（2024 AAHAガイドライン準拠）。旧来の「90mL/kg一括」は廃止
- 猫のショックボーラス: 5〜10mL/kg を15〜20分で投与 → 再評価
- 乳酸値>4mmol/Lは組織低灌流の指標。犬の正常値<2.5mmol/L、猫<2.0mmol/L
- 輸液反応性の評価: ショックインデックス（HR/SBP）＞1.0は低灌流を示唆
- 昇圧薬: ノルエピネフリン 0.1〜2μg/kg/min CRI が第一選択（敗血症性ショック）
- ドパミン 5〜15μg/kg/min CRI。バソプレシン 0.5〜5mU/kg/min CRI（カテコラミン抵抗性に）
- ドブタミン 2〜20μg/kg/min CRI が心原性ショックの第一選択
- 心原性ショックでは大量輸液禁忌。利尿薬＋ピモベンダン＋昇圧薬
- アナフィラキシー: エピネフリン 0.01mg/kg IV/IM → 反応なければ5〜15分後に再投与
- 敗血症: 輸液蘇生に反応不良→1時間以内のノルエピネフリン開始を推奨（surviving sepsis準拠）
- 閉塞性ショック（心嚢液・GDV等）: 原因除去が最優先（心嚢穿刺・胃減圧等）

# 📄 記事3: 猫のHCM（肥大型心筋症）─ 早期発見と管理

## タグ: 循環器 / 猫 / スクリーニング
## 参照論文数: 7本

### 本文の要点:
- HCMは猫の約7頭に1頭（14〜15%）に認められる最も一般的な心臓病（Payne et al. 2015: 780頭の見かけ上健康な猫の調査）
- メインクーン・ラグドールではMYBPC3遺伝子変異が同定（Meurs et al. 2005）
- 診断ゴールドスタンダード: 心エコー（左室壁厚≥6mm）
- ACVIM 2020コンセンサスのステージ分類: A（リスク猫）、B1（軽度・低リスク）、B2（中等度以上・LA拡大）、C（CHF/ATE）、D（治療抵抗性）
- B2の基準: LA/Ao ≥1.6
- B2以上でクロピドグレル18.75mg/日によるATE予防を推奨
- FAT CAT study（Hogan 2015）: クロピドグレルはアスピリンより再発を有意に低下 → クロピドグレルが第一選択
- NT-proBNP: <100pmol/L → 心疾患可能性低い、100〜265pmol/L → グレーゾーン（心エコー推奨）、>265pmol/L → 心疾患可能性高い
- cTnI >0.7ng/mL → 心臓死リスク上昇と独立した関連
- HCM猫の50%以上は聴診で心雑音なし
- LVOTOがない場合のアテノロールの有効性はエビデンス不十分
- ピモベンダン: LVOTOがなく心不全を伴う場合に検討（ACVIM 2020）。閉塞のある場合は注意
- B1でLVOTOありかつ症状あり → アテノロール6.25〜12.5mg PO BID検討
- ステージC: フロセミド1〜2mg/kg IV → 安定後PO BID + ベナゼプリル0.5mg/kg SID + クロピドグレル
- ピモベンダン（収縮能低下時）: 0.1〜0.3mg/kg PO BID（LVOTOなしに限る）
- ATE発症時の疼痛管理: メタドン0.1〜0.3mg/kg IV/IM、フェンタニル1〜5μg/kg/h CRI。NSAIDs禁忌
- ATE時の負荷量: クロピドグレル75mg → 18.75mg/日維持
- 初回ATEの生存退院率: 約40〜50%
- 予後不良因子: 両後肢麻痺、低体温（<37℃）、高K血症、CHF合併
- 再発率: クロピドグレル投与でも年間約24%（FAT CAT study）
- 安静時呼吸数（SRR）: 正常<30回/分（理想<24回/分）。>36回/分持続 → CHF進行疑い
- 日本の注意: クロピドグレルはプラビックス®（75mg錠）を1/4に割って使用。粉砕は苦味が強い

# 📄 記事4: FIP（猫伝染性腹膜炎）─ 抗ウイルス薬時代の治療

## タグ: 感染症 / 猫 / 抗ウイルス薬
## 参照論文数: 7本

### 本文の要点:
- FIPは猫コロナウイルス（FCoV）の変異株による致死的な全身性疾患
- GS-441524の登場で治療成功率は84.6%（2025年 Pathogens誌システマティックレビュー / 11研究650症例）
  - 初回寛解率: 80.6%
  - 再発後の2回目治療：追加4.0%が寛解
- 2024年5月、米国FDAはGuidance #256でGS-441524の調合を事実上容認
- 3病型: 滲出型（60〜70%）、非滲出型（25〜30%）、神経型/眼型（5〜10%）
- 滲出型の診断: 高蛋白滲出液（>3.5g/dL）、A/G比<0.4、Rivalta test陽性（感度〜91%、特異度〜66%）
- FCoV抗体価はFCoV曝露を示すのみ → FIPの確定診断にはならない
- GS-441524はヌクレオシドアナログ。FCoVのRdRpを阻害。レムデシビルは体内でGS-441524に代謝されるプロドラッグ

#### GS-441524 経口投与プロトコル:
- 滲出型/非滲出型（眼・神経なし）: 15mg/kg PO SID × 84日間（12週間）
- 眼型: 15〜20mg/kg PO SID × 84日間
- 神経型: 20mg/kg/日（10mg/kg q12hに分割推奨）× 84日間以上

#### GS-441524 皮下注射プロトコル:
- 滲出型/非滲出型: 6〜7.5mg/kg SC SID
- 眼型: 10mg/kg SC SID
- 神経型: 10mg/kg SC SID（最低量）

- 2024年RCT (Krentz et al.): 滲出型で42日間の短縮プロトコルが84日間と同等の有効性
- 治療開始後24〜48時間で食欲改善、1〜2週間で腹水減少
- 再発は治療完了後2〜8週間に多い
- 日本では動物用医薬品として未承認。英国BOVA社等の海外承認品を動物病院が輸入して使用
- Mutian・Xraphconn等のブラックマーケット製品は品質保証なし
- モルヌピラビルはGS-441524より安価だが、変異誘発のリスクに慎重な意見あり
- 治療総費用: 84日間で約30〜60万円

# 📄 記事5: 犬の免疫介在性血小板減少症（ITP）─ 診断と治療

## タグ: 免疫 / 血液 / 犬
## 参照論文数: 6本

### 本文の要点:
- ITPは犬で最も多い後天性出血障害
- ACVIM 2024ガイドラインに基づく

#### 血小板数と解釈:
- <20,000/μL → ITPを強く支持
- 20,000〜50,000/μL → ITPの可能性あるが他疾患も検討
- 50,000〜100,000/μL → 二次性の検索
- >100,000/μL → 初発診断では原発性ITPをほぼ除外

- 自動血球計算器の血小板数は必ず塗抹で確認（EDTA依存性偽性血小板減少の除外）
- 初期の血小板数は予後予測因子にならない（ACVIM 2024）

#### 治療プロトコル:
- 第一選択: プレドニゾロン 2mg/kg/日 PO（分1〜2）
- 出血がある場合: ビンクリスチン 0.02mg/kg IV 単回 → 血小板回復を加速し入院日数を短縮
- ビンクリスチンはABCB1（MDR1）変異犬種には禁忌（コリー・シェルティ・ボーダーコリー・オーストラリアンシェパード等）
- 治療のゴール: 血小板≥100,000/μLで出血なし

#### 第2免疫抑制薬追加の適応:
- プレドニゾロンに5〜7日で反応なし
- ステロイドの副作用が重篤
- 減薬中の再発
- 25kg以上の大型犬 → 早期併用でステロイド減量を加速（ACVIM 2024推奨）

#### 第2免疫抑制薬の比較:
- MMF: 10〜20mg/kg PO BID、効果発現5〜14日、副作用: 下痢・骨髄抑制
- アザチオプリン: 2mg/kg PO SID（隔日維持）、効果2〜4週、副作用: 骨髄抑制・肝毒性。**猫には禁忌**
- シクロスポリン: 5〜10mg/kg PO BID、効果2〜4週、副作用: 消化器症状・歯肉増殖

#### 減薬プロトコル:
- 血小板≥100,000/μLが2〜4週間安定 → 減薬開始
- 2〜4週間ごとに25%ずつ漸減
- 完全離脱まで通常3〜6ヶ月
- 再発率は約20〜30%
- 脾臓摘出は犬のITPでは推奨されない

#### その他:
- Evans症候群（ITP + IMHA合併）の除外に直接クームス試験
- ITPでは輸注された血小板も即座に破壊 → 効果は一過性
- トラネキサム酸 15〜20mg/kg IV q8h → 出血管理の補助

# 📄 記事6: 犬猫の膀胱炎 ─ ISCAIDガイドライン準拠

## タグ: 泌尿器 / 抗菌薬 / 犬猫共通
## 参照論文数: 6本

### 本文の要点:
- 犬と猫の膀胱炎はまったく異なる疾患概念
- 犬の下部尿路症状のほとんどは細菌性UTI（E.coliが最多: 約50%）
- 若齢〜中齢猫の下部尿路症状の60〜70%は猫特発性膀胱炎（FIC）で細菌感染は1〜2%のみ → 抗菌薬不要
- 高齢猫（10歳以上）ではCKD・DM合併で細菌性UTIが増加

#### 犬の単純性UTI:
- 定義: 解剖学的・機能的異常のない犬に初発〜散発的に起こる細菌性膀胱炎
- 第一選択: アモキシシリン 11〜15mg/kg PO q8〜12h × **3〜5日**（ISCAIDガイドライン 2019）
- 代替: TMP-SMX 15mg/kg PO q12h × 3〜5日
- 代替: アモキシシリン-クラブラン酸 12.5〜25mg/kg PO q12h × 3〜5日
- 治療後のルーチン尿培養は不要（臨床症状の改善が指標）

#### 犬の複雑性UTI:
- 基礎疾患あり: クッシング・DM・CKD・尿路結石等
- 尿培養に基づく抗菌薬選択 × 7〜14日

#### 尿培養の採取と有意菌数:
- 膀胱穿刺（推奨）: ≥10³ CFU/mL
- カテーテル: オス≥10⁴、メス≥10⁵
- 自然排尿（free-catch）: 培養には不適

#### 猫のFIC:
- ヒトの間質性膀胱炎に類似。ストレス応答系の異常が関与
- 膀胱上皮のGAG層の破綻
- 自然寛解しやすい（多くは3〜7日で改善）→ 抗菌薬の誤った「有効性」の印象の原因
- 治療: MEMO（Multimodal Environmental MOdification）
  - 環境の豊かさの改善、トイレの改善（猫数+1個）、ウェットフード中心に変更、ストレス要因の除去
  - 鎮痛: ブプレノルフィン 0.01〜0.02mg/kg 粘膜投与（重症例）

#### 抗菌薬の温存:
- フルオロキノロン → 腎盂腎炎・耐性菌に限定使用
- セフォベシン（コンベニア®）→ ISCAIDは単純性UTIへの第一選択として推奨していない
- 無症候性細菌尿は治療しない（耐性菌増加の原因）

#### 再発性UTI:
- 定義: 12ヶ月に3回以上 or 6ヶ月に2回以上
- 基礎疾患の検索: 結石、クッシング、DM、外陰部解剖異常、CKD、避妊後尿失禁

# 📄 記事7: 犬の椎間板ヘルニア ─ 頚部＆胸腰部統合版

## タグ: 神経 / 整形 / 犬
## 参照論文数: 7本

### 本文の要点:
- Hansen I型（急性脱出: 軟骨異栄養犬種に多い）とII型（慢性突出: 大型犬に多い）
- 最も重要な予後因子: 深部痛覚（DPP）の有無

#### 修正Frankelスケール（胸腰部IVDD）:
- Grade 1: 疼痛のみ → 保存療法
- Grade 2: 不全麻痺（ふらつき歩行可能）→ 保存 or 手術検討、回復率>95%
- Grade 3: 不全麻痺（起立困難）→ 手術推奨、回復率>95%
- Grade 4: 対麻痺（DPP残存）→ 手術、回復率**97〜100%**
- Grade 5: 対麻痺（DPP消失）→ 緊急手術（48時間以内推奨）、回復率**50〜64%**、PMMリスクあり

#### 頚部ヘルニア:
- 好発部位: C2-C3、C3-C4
- 特徴: 激しい頚部痛が主徴。脊柱管が太いため完全麻痺は比較的少ない
- 最重度: 四肢完全麻痺 + 呼吸障害 → 緊急手術

#### 胸腰部ヘルニア:
- 好発部位: T12-T13、T13-L1、L1-L2
- 好発犬種: ダックスフンド、フレンチブルドッグ、ペキニーズ、コーギー、ビーグル
- T3-L3病変 → UMNサイン（反射亢進）
- L4-S3病変 → LMNサイン（反射低下、排尿障害）

#### DPP評価方法:
- 足趾をペアンでクロス圧迫 → 振り向く・鳴く = DPP残存
- 足の引っ込めだけでは不十分（屈曲反射は脊髄反射であり意識的反応ではない）
- 尾のDPP消失はPMMリスク上昇を示唆
- DPP消失が96時間以上 → 回復率47%に低下

#### 術式:
- Ventral Slot（腹側スロット術）: 頚部の標準術式。スロット幅は椎体の1/3を超えない
- Hemilaminectomy（片側椎弓切除術）: 胸腰部の標準術式
- Mini-hemilaminectomy: 脊柱安定性保持の利点
- Dorsal Laminectomy: 多椎間減圧（ウォブラー症候群等）

#### 保存療法:
- 厳格なケージレスト: 4〜6週間
- ガバペンチン 5〜10mg/kg PO BID〜TID
- NSAIDs: メロキシカム 0.1mg/kg PO SID
- **NSAIDs + グルココルチコイドの併用は絶対禁忌**（消化管穿孔リスク）

#### 進行性脊髄軟化症（PMM）:
- DPP消失のGrade 5症例の約2〜10%に発生
- 壊死が頭側・尾側に進行 → 最終的に呼吸筋麻痺で死亡（発症から3〜5日）
- 治療法はない → 安楽死の検討が必要
- 術前にGrade 5の飼い主に必ずIC

#### リハビリ:
- 術後1〜2週: PROM（受動的関節可動域訓練）、マッサージ
- 術後2〜4週: 介助下起立練習、バランスボール
- 術後4〜8週: 水中トレッドミル、陸上歩行
- 術後8週以降: 段階的運動量増加
- 2025年: 機械学習によるGrade 5の予後予測モデル開発中
