# 🦠 FIP（猫伝染性腹膜炎）─ 抗ウイルス薬時代の治療

> ⏱️ **読了時間**: 約5分

#感染症 #猫 #FIP #ウイルス

## 🎯 結論
**「FIP＝不治の病」の常識は過去のものとなりました。** 現在はGS-441524やモルヌピラビル（Molnupiravir）を用いた抗ウイルス療法により、 **80〜90%近い猫で治癒と長期生存が可能** です。治療成功の鍵は、「ウイルスの潜む場所（眼・神経）に合わせた用量設計」と「体重増加に伴う正確な投薬量の増量」です。

## ⚡ 昔の常識 vs 今のエビデンス
| ❌ 旧来の常識 | ✅ 最新のエビデンス |
|---|---|
| FIPと診断されたら、緩和ケアか安楽死を選ぶしかない | **抗ウイルス薬（GS-441524, モルヌピラビル）** により、多くの猫が「治癒」を獲得できる疾患へと変化した。 |
| ドライタイプや神経症状が出たFIPは助からない | 眼や神経へ移行したFIPウイルスには、 **血液脳関門/血液眼関門を突破するための「高用量投与」** で対応可能。 |
| 治療期間は必ず84日間（12週間）必要である | 滲出型の初期・順調な症例では、 **42日間（6週間）への治療期間短縮** も可能とする研究結果（2024年）が出ている。 |

## 📖 詳細解説
### ▼ 🔬 📊 FIPを疑うための診断補助カットオフ値
| 検査項目 | カットオフ値 | 臨床的意義 |
|---|---|---|
| **血清総蛋白** | **> 7.8 g/dL** | 高TP＋高グロブリンの組み合わせでFIPを強く示唆 |
| **A/G比** | **< 0.4** で強く示唆、 **< 0.6** で疑い | FIP鑑別の最も有用な指標の一つ |
| **SAA（血清アミロイドA）** | 高値 | 炎症マーカーとして治療効果のモニタリングにも使用 |

### ▼ 📖 💊 2大抗ウイルス薬の比較と特徴
### ▼ 1️⃣ GS-441524
| 投与経路 | 病型 | 用量 |
|---|---|---|
| **注射（皮下）** | 通常のウェット/ドライ型 | **5〜10 mg/kg** 1日1回 |
| **注射（皮下）** | 神経型/眼型 | **8〜10 mg/kg** 1日1回 |
| **経口** | 通常のウェット/ドライ型 | **10〜15 mg/kg** 1日1回 |
| **経口** | 神経型/眼型 | **15 mg/kg 以上** 1日1回 |

- **成功率** : 全体で約84%。

### ▼ 2️⃣ モルヌピラビル (Molnupiravir)
| 病型 | 用量 |
|---|---|
| 通常 | **10〜15 mg/kg** を **1日2回** 経口 |
| 代替プロトコル | **30 mg/kg** 1日1回 |
| 眼/神経症状あり | **20 mg/kg** を 1日2回 |

- **成功率** : 78%〜90%以上。
- **注意** : 半減期の関係上、 **「1日2回投与」** が基本。

### ▼ 🗺️ 🗺️ 臨床アクション表 ─ FIP治療成功のための要点
| 鉄則 | 具体的なアクション | 失敗に直結するリスク |
|---|---|---|
| **① 用量は「症状の最も重い場所」に合わせる** | 眼の混濁や神経症状があれば **高用量** からスタート。 | 用量不足はウイルスの耐性化と再燃の最大の原因。 |
| **② 体重増加に合わせて薬の量を増やす** | **毎週必ず体重を測り、mg/kgの計算をやり直す** 。 | 相対的な「用量不足」に陥り再燃する。 |
| **③ 治療期間は12週間（84日間）が基本** | 途中中断は耐性ウイルスを生む。 | プロトコル完遂後の再発率は **5〜10%** 。途中中断は再発率を大幅に上昇させる。 |
| **④ 初期の炎症にはステロイド併用** | **プレドニゾロン 1〜2 mg/kg/day** を数日間併用し漸減。 | 長期併用は免疫抑制によりウイルスクリアランスを妨げる。 |

### ▼ 📖 📊 成功率と再発率のまとめ
| 項目 | 数値 |
|---|---|
| 抗ウイルス薬使用時の **治癒・寛解率** | **80〜90%以上** |
| 12週間プロトコル完遂後の **再発率** | **5〜10%** |
| GS-441524の成功率 | 約 **84%** |
| モルヌピラビルの成功率 | **78〜90%以上** |

## 🗣️ 飼い主への説明ガイド
**Q. 「薬がとても高額だと聞きました。途中でやめるとどうなりますか？」**
「FIPの抗ウイルス薬は、ウイルスを完全に叩き切るまで基本は84日間連続投与が必要です。途中でやめると薬に対する『耐性』を持った強いウイルスとして再発してしまう危険が高いです。費用面も含め、最後まで続けられる治療プランを一緒に考えましょう。」

**Q. 「FIPは治る病気になったと聞きましたが、本当ですか？」**
「はい、本当です。適切に治療すれば80〜90%の猫ちゃんが治癒できるようになりました。正しい用量で最後まで投薬を続けることが絶対条件です。治療完了後の再発率は5〜10%程度に抑えられます。」

**Q. 「目がおかしい（濁っている）のですが、それもFIPのせいですか？」**
「はい、FIPのウイルスは眼や脳にも入り込むことがあります。通常よりも高い用量のお薬が必要になりますが、適切に治療すれば回復が期待できます。」
