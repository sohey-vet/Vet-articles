# 🛡️ 免疫介在性溶血性貧血（IMHA）─ 診断と初期治療

> ⏱️ **読了時間**: 約5分

#免疫 #IMHA #救急 #血液

## 🎯 結論
IMHAの治療3本柱は **①免疫抑制（プレドニゾロン                        2mg/kg/日）②抗血栓療法（血栓塞栓症が最大の死因）③輸血** 。重症例（自己凝集や血管内溶血あり）では **初日から** アザチオプリン/シクロスポリン/MMFを併用。最初の2週間の生存がカギ                    ─ 乗り越えた犬の長期予後は良好。再発率は11〜20%。

## 🗺️ IMHA 治療フロー
| ステップ | アクション |
|---|---|
| **1. 安定化** | 輸血（PCV＜15%目安）、酸素化、輸液 |
| **2. 免疫抑制** | プレドニゾロン 2mg/kg/日 <br/> 重症例はDay1から第2免疫抑制薬併用 <br/> 食べない犬はデキサメタゾン IV |
| **3. 抗血栓** | 低分子ヘパリン or クロピドグレル <br/> （血栓塞栓症がIMHA死因の第1位） |
| **4. 再評価** | 48〜72時間でPCV上昇傾向なければ <br/> 二次免疫抑制薬を追加 |
| **5. 漸減** | PCV安定後、3〜6ヶ月かけてゆっくり減量 |

## ⚡ 予後因子
| ⚠️ 予後不良因子 | ✅ 良好な徴候 |
|---|---|
| 血栓塞栓症の併発 | 最初の2週間で安定 |
| 白血球の著明な増加（類白血病反応） | 網状赤血球の増加（再生反応あり） |
| 乳酸値の持続高値 | 免疫抑制に反応してPCV上昇 |
| 基礎疾患（腫瘍・寄生虫）あり | 特発性（基礎疾患なし） |

## 📖 詳細解説
### ▼ 🔬 診断 ─ 自己凝集試験とクームス試験
**要点**
- **球状赤血球（スフェロサイト）** : 血液塗抹で最も手軽に確認できるIMHAの所見
- **スライド凝集試験** : スライドに血液を滴下し生理食塩水で洗浄 → 凝集が残れば陽性
- **クームス試験（直接抗グロブリン試験）** : 赤血球表面の抗体を検出。偽陰性あり
- 併存疾患（薬剤・感染症・腫瘍）の精査が必須（二次性IMHAの除外）

**臨床アクション**
貧血の犬を見たら血液塗抹を確認。球状赤血球＋再生性貧血＋自己凝集陽性ならIMHAの確定診断に十分。

### ▼ 💊 免疫抑制薬 ─ 二次薬の選択
**二次免疫抑制薬の比較**
- **アザチオプリン** : 最も使用実績が長い。効果発現に1〜2週間。肝毒性・骨髄抑制に注意
- **シクロスポリン** : ステロイド反応不良時の代替。腎毒性のモニタリング必要
- **MMF（ミコフェノール酸モフェチル）** : 消化器副作用が犬で出やすい
- **hIVIG（ヒト免疫グロブリン静注）** : 最終手段。高価でアレルギーリスクあり

2024年の幹細胞療法研究では、標準治療に反応不良のIMHAに同種間MSC（間葉系幹細胞）を併用し、76%で血液学的回復を達成。将来の治療選択肢として注目されている。

**臨床アクション**
プレドニゾロン48〜72時間でPCV改善傾向がなければ二次薬追加を検討。「どの二次薬が最良か」はまだ前向きRCT（ランダム化比較試験）のエビデンスなし ─ 施設の経験と患者の状態で判断。

## 🗣️ 飼い主への説明ガイド
**Q. この病気は命に関わりますか？**
はい、重い病気ですが治療法はあります。最初の1〜2週間が最も重要で、この山を越えれば多くの子が回復します。入院治療が必要になる場合もあります。

**Q. 薬の副作用が心配です**
ステロイドは水をたくさん飲む、おしっこが増える、食欲が増すなどの副作用がありますが、命を救うために必要です。状態が安定したら少しずつ減量していきます。
