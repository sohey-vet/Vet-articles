# 手作り食・生食のリスクとエビデンス

> ⏱️ **読了時間**: 約5分

#その他 #栄養

## 🎯 結論
手作り食レシピの **95%がNRC/AAFCO基準を満たしていない** （Stockman et al., 2013）。手作り食を続ける場合、DACVN（獣医栄養学専門医）によるレシピ設計が必須である 生食（RMBD）からはサルモネラ（7〜21%）、リステリア（15〜16%）、カンピロバクター（最大40%）が高頻度で検出され、生食犬は **ESBL産生菌の保菌リスクが5倍以上** 高い **AVMA・AAHA・WSAVAすべてが犬猫への生肉給与に反対** を表明している。飼い主の信念に配慮しつつも、科学的事実に基づいた説明を行うことが臨床現場での責務である

## 📖 詳細解説
### ▼ 📋 栄養欠乏のリスク ─ 「手作り＝安全」の誤解
インターネットや書籍で公開されている犬用手作り食レシピ200種類を解析した研究（Stockman et al., 2013）では、 **95%のレシピ（190/200）がNRCまたはAAFCOの基準を少なくとも1つ満たしていなかった** 。さらに83.5%で複数の必須栄養素（カルシウム、亜鉛、ビタミンD、コリンなど）が欠乏していた。

飼い主の「良いものを食べさせたい」という善意は理解できるが、 **栄養バランスの設計は極めて複雑であり、専門知識なしに達成することは事実上不可能** である。

### ▼ 📋 生食（RMBD）の病原体汚染 ─ One Healthの視点
市販のRMBD（Raw Meat-Based Diets）を対象とした疫学調査では、以下の汚染率が報告されている。

| 病原体 | 汚染率 |
|---|---|
| サルモネラ属菌 | **7〜21%** |
| リステリア菌 | **15〜16%** |
| カンピロバクター属菌 | **最大40%** |

生肉を給与された犬の最大 **30%** が、臨床症状を示さずにサルモネラを糞便中に排泄（シェディング）する。これは同居する **小児、高齢者、免疫不全者** への人獣共通感染症リスクに直結するOne Healthの問題である。

### ▼ 📋 薬剤耐性菌（AMR）の伝播リスク
生食を与えられている犬は、商業用加熱食の犬と比較して **ESBL産生大腸菌の保菌オッズ比が5.0倍以上** である（Baede et al., 2017）。薬剤耐性菌が動物を介して家庭内に広がり、最終的にヒトの治療困難感染症に寄与するリスクは、公衆衛生上の重大な懸念事項である。

### ▼ 📋 生食・手作り食に起因する代表的疾患
**栄養性二次性上皮小体機能亢進症（NSHP）**
- **病態** : カルシウム不足・リン過剰による骨密度低下
- **好発** : 手作り食を与えられた成長期の犬猫
- **診断** : X線での骨透過性亢進、骨皮質菲薄化、折りたたみ骨折。血中iPTHの著明な上昇
- **予後** : 早期にCa:P比を是正（1.2〜1.4:1）すれば数ヶ月で骨密度は回復する。しかし重度例では骨盤狭窄による巨大結腸症や脊椎変形など、 **長期合併症が20〜30%の確率で残存** する

**食事関連拡張型心筋症（Diet-associated DCM）**
- **病態** : BEG（Boutique・Exotic-ingredient・Grain-free）ダイエットや豆類・イモ類主体の手作り食による心拡大・収縮能低下
- **診断** : 心エコーでLVIDd/LVIDs拡大、FS低下。全血タウリン濃度の測定（犬の正常値: >200 nmol/mL）
- **治療** : 食事変更＋タウリン経口補給（犬: **体重あたり 50〜100 mg/kg/day** または **大型犬で500〜1,000 mg PO BID〜TID** ）。心不全がある場合はピモベンダン（0.25〜0.3 mg/kg PO BID）・フロセミド等を併用
- **予後** : 食事変更により3〜6ヶ月で心機能が有意に改善。 **MST（生存期間中央値）が2年に達する** こともあり、最終的に心不全薬の休薬が可能になるケースもある（遺伝性DCMのMSTが数ヶ月〜1年未満であるのと対照的）

**食餌性甲状腺機能亢進症**
- **病態** : 牛・豚の頸部（甲状腺組織を含む気管周辺肉）の生肉給与による外因性甲状腺ホルモン過剰
- **徴候** : 多飲多尿、体重減少、頻脈、パンティング
- **診断** : 総T4 > 4.0 μg/dL、遊離T4上昇、TSH低下。甲状腺腫瘤なし
- **治療** : 原因生肉の給与中止のみ。抗甲状腺薬は不要
- **予後** : 給与中止後 **2〜4週間以内に正常化** （予後良好）

### ▼ 💊 臨床アプローチ ─ 推奨治療アルゴリズム
1. **総合栄養食への切り替え** : AAFCO給与試験クリア、またはWSAVA推奨基準（フルタイムの獣医栄養学専門医在籍等）を満たすメーカーの製品を推奨

2. **手作り食の継続を強く希望する場合** : DACVN（米国獣医栄養学専門医）またはECVCN専門医によるレシピ設計を必須とし、3〜6ヶ月毎の定期血液検査およびBCS/MCS評価を実施

3. **細菌性胃腸炎・敗血症** : 対症療法を主体とし、 **盲目的な広域抗菌薬投与はAMR防止の観点から厳格に避ける** 。敗血症基準を満たす重症例のみ培養・感受性試験に基づいた抗菌薬治療を行う

### ▼ 📋 日本の臨床実情への配慮
- 日本では「ペットに良いものを食べさせたい」という飼い主意識が強く、手作り食・生食の実践率は欧米以上に高い傾向がある
- DACVN/ECVCN認定の獣医栄養学専門医は国内に極めて少なく、 **オンラインの栄養相談サービス** （BalanceIT®等）の活用が現実的な選択肢である
- 生食のサルモネラリスクは、日本の鶏肉（カンピロバクター汚染率が高い）においても同様に注意が必要である

## 🗣️ 飼い主への説明ガイド
**Q. 「手作りごはんのほうが市販のフードより体にいいですよね？」**
お気持ちはよくわかります。ただ、研究では市販の手作り食レシピの95%が必要な栄養素を満たしていないことがわかっています。特にカルシウムやビタミンDなどが不足しやすく、長期間続けると骨がもろくなるなどの問題が起きることがあります。もしどうしても手作り食をご希望でしたら、動物の栄養専門医がレシピを設計するサービスがありますので、そちらをご利用いただくのが安全です。

**Q. 「生肉を食べさせるのは自然だから安全ではないですか？」**
生肉にはサルモネラ菌やカンピロバクターなどの食中毒菌が含まれていることが多く、ワンちゃん・猫ちゃん自身は元気でも、うんちの中に菌を出し続けてしまうことがあります。特にお子さんやご高齢の方、免疫が弱い方がいらっしゃるご家庭では感染のリスクがあります。世界的な獣医師の団体も生食には反対の立場をとっていますので、加熱したフードをおすすめしています。

**Q. 「グレインフリーのフードのほうが良いと聞きましたが？」**
最近の研究では、グレインフリー（穀物不使用）のフードと心臓病（拡張型心筋症）の関連が報告されています。穀物アレルギーが確定している場合を除き、あえてグレインフリーを選ぶ必要はありません。ごはん選びで迷われたら、いつでもご相談ください。
