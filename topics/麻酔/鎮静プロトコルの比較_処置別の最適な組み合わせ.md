# 😴 鎮静プロトコルの比較 ─ 処置別の最適な組み合わせ

> ⏱️ **読了時間**: 約5分

#麻酔 #鎮静

## 🎯 結論
鎮静プロトコルは「処置の痛みレベル」「所要時間」「患者のリスク」で選択する。犬の軽度鎮静（画像検査等）には **Butorphanol 0.2-0.4 mg/kg +                        Acepromazine 0.01-0.03 mg/kg IV/IM** 。中等度以上の処置には **Dexmedetomidine 3-5 mcg/kg                        IV（またはIM 5-10 mcg/kg）+ Butorphanol 0.2 mg/kg** が主軸。猫の難しい症例には **「Kitty                        Magic」（Dexmedetomidine 10-20 mcg/kg + Ketamine 3-5 mg/kg + Butorphanol 0.2-0.3 mg/kg IM                        混合1ショット）** で15-20分の不動化が得られる。来院前の不安軽減には **Gabapentin 50-100 mg/cat                        PO** を2-3時間前に投与。心疾患猫では循環への影響が少ない **Alfaxalone 2-3 mg/kg IM + Butorphanol 0.2 mg/kg                        IM** が使いやすい。

## 📋 犬の鎮静チートシート
| 処置レベル | 薬剤 | 用量 |
|---|---|---|
| **軽度** <br/> (X線、エコー、採血) | Butorphanol <br/> ± Acepromazine | **0.2-0.4 mg/kg** IV/IM <br/> **0.01-0.03 mg/kg** IV/IM |
| **中等度** <br/> (創傷処置、短時間処置) | Dexmedetomidine <br/> + Butorphanol | **3-5 mcg/kg IV** or **5-10 mcg/kg IM** <br/> 0.2 mg/kg IV/IM |
| **深い鎮静** <br/> (CT、MRI、骨髄穿刺) | Dexmedetomidine <br/> + Butorphanol <br/> + Midazolam | **5-10 mcg/kg IM** <br/> 0.2 mg/kg <br/> **0.2 mg/kg** IV/IM |

## 📋 猫の鎮静チートシート
| 処置レベル | 薬剤 | 用量 |
|---|---|---|
| **来院前・不安軽減** | Gabapentin | **50-100 mg/cat PO** （来院2-3時間前） |
| **軽度** <br/> (採血、X線) | Butorphanol | **0.2-0.4 mg/kg IM** |
| **中〜深い鎮静** <br/> (「Kitty Magic」) | Dexmedetomidine <br/> + Ketamine <br/> + Butorphanol | **10-20 mcg/kg IM** <br/> **3-5 mg/kg IM** <br/> **0.2-0.3                                        mg/kg IM** <br/> （混合1ショット） |
| **心疾患猫** <br/> (Alfaxaloneプロトコル) | Alfaxalone <br/> + Butorphanol | **2-3 mg/kg IM** <br/> 0.2 mg/kg IM |

## 📖 詳細解説
### ▼ 💉 Dexmedetomidine ─ 中心的な鎮静薬の使い方
**特徴**
- α₂アゴニスト。強力な鎮静・鎮痛作用。用量依存的
- 犬: **3-5 mcg/kg IV** （中等度） or **5-10 mcg/kg IM**
- 猫: **10-20 mcg/kg IM** （「Kitty Magic」の一成分）

**拮抗薬**
- **Atipamezole (Antisedan)** : Dexmedetomidineと等量（volume）IM
- 処置終了後に使えばスムーズな覚醒が得られる

**⚠️ 注意が必要な患者**
- **心疾患** : 徐脈・末梢血管収縮 → 心負荷増大。HCM猫では特に注意
- **高齢・衰弱** : 低用量から開始（健康な若い動物の半量以下）
- **嘔吐** : 特に猫で嘔吐を誘発しやすい → 絶食確認

### ▼ 🐱 Alfaxalone ─ 心疾患猫の味方
**猫での使い方**
- **Alfaxalone 2-3 mg/kg IM + Butorphanol 0.2 mg/kg IM**
- 循環への影響が少なく、 **心疾患（HCM等）の猫でも使いやすい**
- 欠点: 投与容量が多くなりやすい

**犬での使い方**
- 犬ではAlfaxalone **単独IM投与は推奨されない** （興奮のリスク）
- 必ず **Midazolam 0.2 mg/kg** やButorphanolと併用する

### ▼ 📊 鎮静中のモニタリング ─ 最低限押さえるべき指標
- **SpO2** : パルスオキシメーター装着。94%以上を維持
- **心拍数** : α₂アゴニスト使用時は徐脈を監視
- **呼吸数・パターン** : 無呼吸・低換気に注意
- **粘膜色・CRT**
- **酸素投与** : Flow-by酸素を推奨（特に深い鎮静時）
