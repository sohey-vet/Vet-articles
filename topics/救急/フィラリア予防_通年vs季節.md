# フィラリア予防 ─ 通年vs季節

## 🎯 結論

- **AHS（米国糸状虫学会）・CAPCは「通年予防」を例外なく推奨**している。季節予防の最大リスクは飼い主の投与忘れ（非コンプライアンス）であり、感染症例の約80%以上がこれに起因する
- 予防薬の投与歴にかかわらず**年1回の抗原＋MF検査**が必須。偽陰性（約7%）には血清の**熱処理**による感度向上が有効
- 成虫駆除はAHS推奨の**3-dose protocol**（ドキシサイクリン前投与＋メラルソミン3回）が合併症を最小化し駆除率**98%以上**を達成する

## 本文

### 通年予防 vs 季節予防 ─ なぜ通年なのか

#### 気候変動の影響
- 温暖化・ヒートアイランド現象により蚊の活動期間が延長
- 外気温14℃（57℉）超でMF（ミクロフィラリア）の体内発育が進行
- **室内、地下街、温水プール近くなどの微小気候（Microclimate）**で冬期でも感染サイクルが成立するリスクがある

#### コンプライアンスの問題 ─ 最大のリスク因子
- 季節予防における飼い主の投与忘れ（開始の遅れ、終了の前倒し）が最大の感染リスク
- 予防薬投与犬の感染症例のうち、**約80%以上が非コンプライアンス**に起因

#### One Healthの視点
- 大環状ラクトン系予防薬（ミルベマイシン等、またはイベルメクチン・ピランテル合剤）は消化管内線虫（犬回虫、犬鉤虫）の駆除効果も持つ（※フィラリア予防用の低用量イベルメクチン単剤では線虫駆除はできないため注意）
- **通年投与によりトキソカラ症や皮膚幼虫移行症**などの人獣共通感染症リスクを年間通じて低減できる

---

### 診断

年1回のスクリーニングがすべての犬に推奨される。

#### 抗原検査（ELISA・免疫クロマトグラフィー）
- 雌成虫の生殖器由来抗原を検出
- **特異度 >95%、感度 79〜90%以上**（成虫3隻以上で感度向上）
- 抗原抗体免疫複合体による**偽陰性が最大約7%**存在 → **血清の熱処理**（104℃のヒートブロックで10分加熱後、遠心分離して得た上清を使用する専用プロトコル等）により感度が有意に上昇

#### MF検査（Knott法・直接塗抹法）
- 抗原検査と**必ず併用**
- MF陽性・抗原陰性の場合: 免疫複合体、未成熟成虫、または他種フィラリアの可能性

#### Class分類

| Class | 臨床像 |
|-------|--------|
| 1（無症状〜軽度） | 臨床症状なし〜時折の咳嗽 |
| 2（中等度） | 運動不耐性、右肺動脈拡張 |
| 3（重度） | 呼吸困難、右心不全（腹水）、悪液質 |
| **4（Caval Syndrome）** | **虚脱、血色素尿、三尖弁逆流 ─ 致死的緊急事態** |

---

### 予防薬（月1回投与・通年推奨）

| 薬剤 | 用量 | 投与経路 |
|------|------|----------|
| イベルメクチン | **6〜12 μg/kg** | PO |
| ミルベマイシンオキシム | **0.5 mg/kg** | PO |
| モキシデクチン | **3 μg/kg** PO / 2.5 mg/kg Spot-on | PO・Spot-on・注射 |
| セラメクチン | **6 mg/kg** | Spot-on |

**「Double Defense」アプローチ**: 薬剤耐性株（JYD-34株など）への対策として、大環状ラクトン＋ピレスロイド系外部寄生虫駆除薬の**通年併用**が最新トレンド

---

### 成虫駆除 ─ AHS 3-dose Protocol

1. **Step 1**: 診断直後から通年予防薬投与開始（MFの漸減）
2. **Step 2**: `ドキシサイクリン` **10 mg/kg PO BID, 4週間** → 共生菌*Wolbachia*を死滅させ虫体を弱体化。PTE（肺血栓塞栓症）リスクも低下。その後1ヶ月休薬
3. **Step 3**: `メラルソミン二塩酸塩` 深部筋肉内注射（脊柱起立筋）
   - 第1回: **2.5 mg/kg IM**
   - 1ヶ月後 第2回: **2.5 mg/kg IM**
   - 24時間後 第3回: **2.5 mg/kg IM**
4. **厳格な運動制限**: 最後の注射後**6〜8週間**ケージレスト

**Caval Syndrome（Class 4）**: 右頸静脈からフレキシブルアリゲーター鉗子を挿入し透視下で虫体を物理的に摘出

---

### 予後

| 状況 | 予後 |
|------|------|
| 通年予防の厳守 | 感染率**ほぼ0%** |
| 3-dose protocol | 成虫駆除率**98%以上** |
| 重篤なPTE | ドキシサイクリン前投与＋運動制限で**数%未満**に抑制 |
| Caval Syndrome（未手術） | 致死率**ほぼ100%**（MST: 数日） |
| Caval Syndrome（外科的摘出成功） | 生存率（退院率）**約50〜70%** |
