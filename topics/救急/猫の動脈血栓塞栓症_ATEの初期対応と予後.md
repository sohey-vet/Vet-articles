# 🚨 猫の動脈血栓塞栓症（ATE）─ 初期対応と予後

> ⏱️ **読了時間**: 約5分

#救急 #循環器 #猫 #心筋症

## 🎯 結論
猫のATEは極めて強い疼痛を伴うため、来院直後に **最優先で強力な鎮痛（Fentanyl Bolus 2-5 mcg/kg IV                        等）** を行う（Buprenorphineでは不十分なことが多い）。診断は **The 5 Ps** （Pain, Paresis/Paralysis,                    Pulselessness, Pallor, Poikilothermy）で行う。急性期の抗血栓療法として **未分画ヘパリン（UFH）250-300 IU/kg IV後、150-250                        IU/kg SC                        q6-8h** （またはLMWH）を開始。血栓溶解療法（tPAなど）は再灌流障害リスクから一般に推奨されない。慢性期・再発予防には **Clopidogrel 18.75                        mg/cat PO q24h** が第一選択（Rivaroxabanとの併用＝Dual                    therapyの報告も増加）。全体の生存退院率は **30-40%** と厳しく、特に **直腸温 < 37.0℃、徐脈、両後肢完全麻痺** は予後不良の徴候。心筋症（HCM等）の併発評価も必須。

## 🗺️ 薬剤プロトコル
| カテゴリ | 薬剤 | 用量・投与法 |
|---|---|---|
| **鎮痛（最優先）** | Fentanyl | Bolus: 2-5 mcg/kg IV <br/> CRI: 2-5 mcg/kg/hr |
| **鎮痛（代替）** | Methadone | 0.1-0.2 mg/kg IV/IM q4-6h |
| **抗血栓（急性期）** | 未分画ヘパリン (UFH) | 初回: 250-300 IU/kg IV <br/> 維持: 150-250 IU/kg SC q6-8h |
| **抗血栓（維持）** | Clopidogrel | **18.75 mg/cat PO q24h** （第一選択） |
| **抗血栓（オプション）** | Rivaroxaban | 2.5 mg/cat PO q24h（Clopidogrelと併用可） |

| ❌ 旧来 | ✅ 最新 |
|---|---|
| とりあえずレントゲンを撮る | **鎮痛が先** 。極度の疼痛とパニックにより突然死するリスクあり |

| 積極的に血栓を「溶かす」治療 | tPAなど血栓溶解薬の使用は、カリウムや乳酸の急激な全身還流（ **再灌流障害** ）により致死的になりやすい |

## 📖 詳細解説
### ▼ 🩺 診断: The 5 Ps の確認
- **Pain (痛覚/疼痛)** : 初期は極度の疼痛で泣き叫ぶ。
- **Paresis/Paralysis (不全麻痺/麻痺)** : 安静時の脱力。下位運動神経障害の所見。
- **Pulselessness (脈拍消失)** : 大腿動脈の触知不可。ドップラーで血流音消失。
- **Pallor (蒼白)** : 肉球や爪床が青白～紫色。健康な肢と比べる。
- **Poikilothermy (変温/冷感)** : 患肢が明らかに冷たい。

### ▼ 📉 予後不良サイン ─ 安楽死の判断基準
**生存を大きく低下させる要因**
- 来院時の **直腸温 < 37.0℃**
- 徐脈の存在（ショック・高カリウム血症の示唆）
- **両後肢** の完全な運動機能喪失（鞍状血栓）
- 前肢など複数部位での同時塞栓
- 重度うっ血性心不全（肺水腫）の併発

**💡 臨床Tips**
- 初診時に予後を数字（30-40%の生存退院率）とともに伝え、高額な治療費と高い再発率（25-50%）があることを説明する。
- 苦痛を取り除くことができない、または後肢の壊死が進行する場合、安楽死は妥当な選択肢となる。

## 🗣️ 飼い主への説明ガイド
**Q. 治る見込みはありますか？**
発見が早く、片脚のみの軽度な麻痺であれば歩けるようになる可能性もありますが、両脚が完全に麻痺して体温が低い場合は非常に危険な状態（生存率30%以下）です。また心臓の重い病気が原因であるため、血栓が溶けても急な心不全や心停止のリスクが常に伴います。

**Q. すぐに手術で血栓を取れませんか？**
猫の場合、血管が非常に細く、全身状態が極めて悪いため手術（血栓摘出術）は通常推奨されません。お薬で血栓の広がりを抑えながら、強い痛みを取り除き、心臓を休ませる治療が中心となります。
