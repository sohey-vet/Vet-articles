🎯 結論

ショックは4類型（循環血液量減少性・血液分布異常性・心原性・閉塞性）に分類され、治療アプローチが根本的に異なる。 最初の1時間（Golden Hour）の対応が生死を分ける。 2024 AAHA輸液ガイドラインでは、循環血液量減少性ショックへのボーラス量として犬15〜20mL/kg、猫5〜10mL/kgを15〜30分で投与し、灌流パラメータを再評価して追加投与を判断する。 心原性ショックでは輸液は禁忌に近い ─ 間違えれば肺水腫を悪化させる。 輸液蘇生に反応しない低血圧にはノルエピネフリンが第一選択の昇圧薬（DACVECC専門医の87.9%が犬で第一選択として使用）。

原因と病態生理

  • 出血性: 外傷、脾臓破裂、消化管出血、凝固異常、術後出血
  • 非出血性: 重度の嘔吐/下痢、多尿（糖尿病・腎不全）、熱傷、第三空間貯留（腹水・胸水）
  • 病態: 前負荷↓ → 心拍出量↓ → 組織灌流↓ → 代償性頻脈・末梢血管収縮

輸液蘇生プロトコル（2024 AAHA GL準拠）

項目 犬 猫
初回ボーラス 15〜20 mL/kg 5〜10 mL/kg
投与速度 15〜30分で投与
輸液の種類 等張平衡晶質液（乳酸リンゲル推奨。生食は高Cl性代謝性アシドーシスのリスク）
再評価 ボーラス後に灌流パラメータ再評価 → 改善なければ追加ボーラス

（ただし一括投与はしない）		 40〜60 mL/kg/h

💊 臨床Tips

  • 出血性ショックで2〜3回のボーラスに反応しない場合 → pRBC（濃厚赤血球）の準備を開始
  • 閉鎖腔出血（血胸・血腹）: 大量輸液は出血を加速させるリスクあり → 小容量蘇生（permissive hypotension）を考慮
  • 低蛋白血症が並存する場合: コロイド（HES 3〜5mL/kg ゆっくり投与）の併用を検討。ただしAKIのリスクを考慮

敗血症性ショック

  • 全身性炎症反応による血管拡張 + 毛細血管透過性亢進 → 「相対的な」循環血液量不足
  • 初期（暖かいショック）: 粘膜紅潮・バウンディングパルス・CRT短縮・発熱
  • 後期（冷たいショック）: 血管収縮・粘膜蒼白・低体温 → 区別が循環血液量減少性と困難に

💊 治療の3本柱

  1. 輸液蘇生: まず晶質液ボーラスで前負荷を確保（犬15-20mL/kg、猫5-10mL/kg）
  2. 原因感染の治療: 広域抗菌薬を診断後1時間以内に投与（SSC 2021の推奨をヒトから外挿）
  3. 昇圧薬: 2〜3回のボーラスでMAP 65mmHg（犬）を達成できなければノルエピネフリンCRI開始

アナフィラキシーショック

  • 犬のアナフィラキシーの標的臓器は肝臓（門脈のうっ血） → 嘔吐・下痢・血便が主症状（ヒトの気道浮腫とは異なる）
  • 猫のアナフィラキシーは呼吸器症状（気管支攣縮・呼吸困難）が主体
  • 第一選択: エピネフリン 0.01mg/kg IM（1:1000 製剤）。IV投与は不整脈のリスクがあるため慎重に
  • 輸液蘇生 + デキサメタゾン 0.1〜0.2mg/kg IV + ジフェンヒドラミン 1〜2mg/kg IM

⚠️ 日本の注意: ジフェンヒドラミン注射液は日本では動物用として流通していないため、レスタミン錠の粉砕 PO や、クロルフェニラミン0.2〜0.5mg/kg IV/IM で代用することが多い。

なぜ輸液が禁忌なのか

  • 心原性ショック = ポンプ不全 → すでに前負荷が過剰（Frank-Starlingカーブの右端）
  • 輸液を追加 → うっ血悪化 → 肺水腫 → 呼吸不全で状況がさらに悪化
  • 「低血圧だから輸液」という反射的な対応は心原性を見逃すと致命的

💊 鑑別のポイント

  • 聴診: ギャロップリズム・心雑音の有無
  • 胸部X線: 心拡大・肺静脈怒張・肺水腫像
  • POCUS（ポイントオブケア超音波）: LA/Ao比、心嚢液の有無 → 数分で鑑別可能
  • ⚠️ ショック状態の動物にX線撮影は負荷が大きい → POCUSが使えれば優先

治療

  • 酸素投与: フローバイ or 酸素ケージ
  • フロセミド: 肺水腫を伴う場合 1〜2mg/kg IV（利尿開始まで20〜30分）
  • ドブタミン: 2.5〜10μg/kg/min CRI（正のイノトロープ。心拍出量を改善）
  • ピモベンダン: 0.15〜0.3mg/kg PO BID（ベトメディン注がある場合はIV 0.15mg/kg）
  • 不整脈が原因の場合 → 抗不整脈薬（リドカイン for VT、アトロピン for AV block等）

心タンポナーデ

  • 心嚢内の液体貯留により心室拡張が制限 → 心拍出量↓
  • 身体所見: 心音減弱、頸静脈怒張、奇脈（吸気時に収縮期血圧が10mmHg以上低下）
  • POCUS: 心嚢液の確認は超音波で数秒。心エコーで右房・右室の虚脱を確認
  • 心嚢穿刺（pericardiocentesis）が救命処置 → 少量の排液でも劇的に改善

猫の動脈血栓塞栓症（ATE）

  • 主にHCMに続発。左心耳内血栓が腹部大動脈分岐部に塞栓
  • 5P: Pain, Pulselessness, Pallor, Paralysis, Poikilothermia（患肢の冷感）
  • 疼痛管理が最優先（メタドン 0.1〜0.3mg/kg IV/IM or フェンタニル CRI）
  • クロピドグレル 18.75mg PO 初回負荷 → 18.75mg/日 維持（FAT CAT study: クロピドグレルはアスピリンより再発率を有意に低下）
  • 予後: 初回発症の生存退院率は約40〜50%。両後肢麻痺 + 低体温 + 高K血症は予後不良因子

昇圧薬の適応

輸液蘇生（晶質液ボーラス2〜3回）でMAP≥65mmHg（犬）を達成できない場合に開始。

（第一選択）		 0.1〜2.0 μg/kg/min

β1軽度 → 心収縮力↑		 DACVECC専門医の87.9%（犬）・83.1%（猫）が第一選択。ドパミンより不整脈リスクが低い

💊 臨床Tips

  • 昇圧薬は必ず中心静脈カテーテルまたは専用の末梢ラインから投与。血管外漏出は組織壊死を起こす
  • 日本の一般病院ではノルエピネフリンの在庫がないことも → ドパミンで開始し、可能なら早期にNEに切替が現実的
  • 血圧モニタリング: ドップラー or オシロメトリック → MAPをリアルタイムで追跡
  • ラクテートのトレンドが輸液蘇生の有効性の最良指標。2時間ごとの再測定を推奨

💊 乳酸値の読み方

  • ＜2.5 mmol/L: 正常範囲
  • 2.5〜4.0 mmol/L: 軽度〜中等度の組織低灌流
  • ＞4.0 mmol/L: 重度ショック → 積極的蘇生が必要
  • ＞8.0 mmol/L: 予後不良因子
  • トレンドが重要: 治療開始後2〜6時間で30%以上の低下がなければ治療効果不十分

ショック状態とは何ですか？

ショックとは、体中の臓器に十分な血液が行き渡らなくなっている状態です。出血・感染症・心臓の問題など、原因はさまざまです。放置すると臓器が傷ついてしまうため、できるだけ早く治療を開始することが大切です。

治療費はどのくらいかかりますか？

ショックの治療は集中治療が必要なため、一般的に入院費・検査費・薬剤費を含めて5万〜20万円/日程度かかることが多いです。昇圧薬の持続投与が必要な場合はさらに費用がかかります。事前にお見積もりをお出しいたします。

生存率はどのくらいですか？

原因と重症度によって大きく異なります。外傷性の出血ショックで早期に治療を開始できた場合は回復が見込めますが、敗血症性ショックやDICを合併した場合は予後が厳しくなります。治療開始後数時間の反応を見て、改めてお話しいたします。
