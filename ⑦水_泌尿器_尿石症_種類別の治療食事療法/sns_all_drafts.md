## 🧵 Threads用（長文・専門的）

```text
最新のACVIMコンセンサスおよびMUC推奨は、尿石症管理における診断・治療戦略に新たな指針。

犬のストルバイト結石はウレアーゼ産生菌感染に起因する場合が多く、適切な抗菌療法と溶解食の併用が必須です。画像検査で結石が消失しても、核内に細菌が残存する可能性を考慮し、抗菌薬の一定期間継続が推奨されます。

猫のストルバイトは95%以上が無菌性病態を呈するため、原則として抗菌薬は不要。溶解食単独での管理が有効です。溶解期間内に縮小傾向が認められない場合、混合結石の鑑別診断が必要となります。

シュウ酸カルシウム結石は食事による溶解が不可能。物理的除去後の再発予防が治療の要です。尿比重目標値の達成、厳格な尿pH管理（6.5〜7.5）、高再発例におけるチアジド系利尿薬の考慮、原発性上皮小体機能亢進症の除外診断も重要項目です。

尿酸塩結石においては、アロプリノール療法と低プリン食の厳密な併用が、キサンチン結石形成リスク回避に不可欠。シスチン結石は遺伝的素因を背景とし、低蛋白食と尿アルカリ化が基本的なアプローチ。難治例ではチオプロニン投与も考慮されます。

いずれの結石においても、摘出後の成分分析に基づく個別化された再発予防戦略の確立が不可欠です。

#獣医腎泌尿器科 #結石病態
```

結石管理の結論 ─ 種類ごとのアプローチ
・ストルバイト：溶解可能（犬は抗菌薬必須、猫は原則不要）
・シュウ酸Ca：溶解不可（外科的除去＋再発予防が命）

[3枚目: 結石の種類と特徴（一覧表）]
頻度と特徴の整理
◆ストルバイト（犬40%/猫50%）
X線：陽性 / 溶解：可能 / 関連：犬はUTI・猫は無菌性
◆シュウ酸Ca（犬40%/猫40%）
X線：陽性 / 溶解：不可 / 関連：高Ca血症・酸性尿
◆尿酸塩（犬5〜8%）
X線：陰性〜弱陽性 / 溶解：可能 / 関連：ダルメシアン・PSS
◆シスチン（犬1〜3%）
X線：弱陽性 / 溶解：困難 / 関連：遺伝性（マスティフ等）

[4枚目: 犬のストルバイト（感染誘発性）]
犬のストルバイト管理の基本
・原因：ウレアーゼ産生菌（Staphylococcus, Proteus等）によるアルカリ尿
・治療：適切な抗菌薬 ＋ 溶解食（感染を治療しないと溶けない）
・モニタリング：4週ごとにX線でサイズ確認
・臨床Tips①：結石がX線で見えなくなってから、さらに1ヶ月抗菌薬を継続する
・臨床Tips②：尿培養陰性でも、結石核内に細菌が封入されている可能性あり

[5枚目: 猫のストルバイト（無菌性）]
猫のストルバイト管理の基本
・原因：95%以上が無菌性 → 抗菌薬は原則不要！
・治療：溶解食（尿酸性化＋Mg/P制限）のみで対応
・期間：約2〜5週間で溶解
・外科適応：4〜6週で縮小傾向がなければ、混合結石を疑い外科的除去を検討
・予防：水分摂取量増加（ウェットフード推奨）＋酸性化フードへ移行

[6枚目: シュウ酸Ca結石（物理的除去）]
シュウ酸Caの管理 ─ 溶解不可
・膀胱切石術：最も一般的。術後X線で残存結石がないことを確認
・ボイディングウロハイドロプロパルション：尿道径より小さい小結石を非侵襲的に排出
【日本での留意点】
・犬：高Ca血症の精査（イオン化Ca測定）。原発性上皮小体機能亢進症が潜んでいるケースに注意
・猫：尿管結石の増加。SUBシステム（皮下尿管バイパス）の適応拡大

[7枚目: シュウ酸Caの再発予防（MUC 2024推奨）]
再発予防の具体的目標と介入
・水分摂取：目標 尿比重 犬 ＜1.020 / 猫 ＜1.030
・尿pH：目標 6.5〜7.5（酸性を避ける）
　→ 必要時 クエン酸カリウム 75 mg/kg PO BID
・食事制限：高Ca食品（チーズ・ヨーグルト）、高シュウ酸食品（ほうれん草等）、Ca単体サプリは避ける
・難治例（犬）：ヒドロクロロチアジド 1〜2 mg/kg PO BID

[8枚目: その他の結石（尿酸塩・シスチン）]
特殊な結石へのアプローチ
◆尿酸塩（ダルメシアン、肝門脈シャントに好発）
・基本：低プリン食＋尿アルカリ化＋水分摂取増加
・溶解：アロプリノール（犬 5〜7 mg/kg PO BID または 15 mg/kg SID）
※注意：アロプリノール単独はキサンチン結石形成リスク大！必ず低プリン食と併用する。
◆シスチン（ニューファンドランド等に好発）
・基本：低蛋白食＋尿アルカリ化。難治例はチオプロニン

[9枚目: まとめ]
本記事の要点
1. ストルバイトの溶解戦略：犬は「抗菌薬＋溶解食」、猫は「溶解食のみ」が基本。犬は消失後も1ヶ月抗菌薬を継続する。
2. シュウ酸Caは溶解不可：物理的除去後、尿比重の低下（犬＜1.020 / 猫＜1.030）と尿pH（6.5〜7.5）の管理で再発を徹底予防する。
3. 尿酸塩に対するアロプリノール等の薬剤使用時は、副作用（キサンチン結石等）に要注意。摘出結石は必ず成分分析へ提出する。
