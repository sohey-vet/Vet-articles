### 3. 投稿フォーマット（テンプレート）

```
⚡ 昔の常識 vs 今のエビデンス

❌ 昔: [旧来の常識]
✅ 今: [最新のエビデンス]

明日の診療から変えられます 💪

#獣医 #獣医師 #エビデンス #[テーマタグ]
```

#### ② 「結論ファースト」型
```

[リンク]
```

#### ③ 「数字で驚かせる」型
```

[補足説明1〜2文]

```

#### ④ 「症例クイズ」型
```
🐾 症例クイズ

[犬種/猫種]、[年齢]、[主訴]

#獣医クイズ #獣医師
```

### 4. 成長戦略（フォロワー獲得）

#### Phase 1: 0 → 500フォロワー（1〜2ヶ月）
- **毎日1投稿**（最低限）、できれば朝7時 + 夜20時の2回
- **既存の獣医系アカウントに積極的にリプ＆引用RT**
- **「昔 vs 今」形式**を多用（議論が生まれやすい）
- ハッシュタグは5〜8個（多すぎるとスパム認定される）
- **獣医系インフルエンサーの投稿に有益なリプを付ける**（寄生型成長）

#### Phase 2: 500 → 2,000フォロワー（3〜4ヶ月）
- 症例クイズを週1回導入（エンゲージメント率UP）
- **スレッド投稿**（1つのテーマを5〜8ツイートで深掘り）
- 画像カードの導入（Canvaでテンプレ作成）
- 他のアカウントとのコラボ企画

#### Phase 3: 2,000 → 5,000+フォロワー（6ヶ月〜）
- Noteの有料記事告知を本格化
- Xスペースで「ライブ論文抄読会」（月1回）
- フォロワー参加型コンテンツ（アンケート・投票）

### 5. 投稿の最適時間

| 時間帯 | ターゲット | おすすめ |
|--------|-----------|---------|
| **7:00〜8:00** | 出勤前の獣医師 | ⭐ 最高 |
| 12:00〜13:00 | 昼休み | ◎ |
| **20:00〜22:00** | 仕事後のリラックスタイム | ⭐ 最高 |
| 23:00〜 | 夜勤中の獣医師 | △ ニッチだが刺さる |

### 6. やってはいけないこと

- ❌ 宣伝ばかりの投稿（Note告知は週1〜2回に抑える）
- ❌ 一般飼い主向けの投稿（ターゲットがブレる）
- ❌ 診断・治療の断定（「○○すべき」→「○○が推奨されている」）
- ❌ 根拠のない主張（必ず論文を付ける）
- ❌ 他の獣医師の治療方針を批判（「違うアプローチもある」程度に）
- ❌ 投稿を何日も空ける（毎日が理想、最低でも週5）

### 2. コンテンツ構成

#### 🆓 無料コンテンツ（Phase 1: 2月〜5月）

**全文無料で公開する記事**:
- 結論（30秒で読める）
- 昔の常識 vs 今のエビデンス表
- 臨床アクション表
- 詳細解説
- 飼い主説明ガイド
- 参照論文

**目的**: 信頼構築。「ここは本物だ」と認知してもらう。

#### 💰 有料化（Phase 2: 6月〜）

**無料で残す部分**（ここだけで「役に立った」と感じられる）:
- 結論（1〜2文）
- 昔 vs 今の比較表
- 臨床アクション表（重症度別）
- 日本の臨床実情（簡易版）

**有料にする部分**（「これは金を払う価値がある」と思える）:
- 📖 詳細解説（論文の中身を噛み砕いた深掘り）
- 🗣️ 飼い主への説明ガイド（そのまま使えるスクリプト）
- 💰 製品比較＋コスト表
- 📚 参照論文リスト（DOIリンク付き）
- 💡 臨床Tips（経験則＋エビデンス）

### 3. 価格設定

| 形式 | 価格 | 位置づけ |
|------|------|---------|
| **単品記事** | 300〜500円 | お試し購入・衝動買い |
| **マガジン（月額）** | 980円/月 | メインの収益源 |
| **マガジン（年額）** | 9,800円/年（17%OFF） | ロイヤルユーザー向け |
| **全記事バンドル** | 4,980円（買い切り） | 将来の選択肢 |

#### なぜ980円/月か？

- 獣医系セミナー = 1回 5,000〜30,000円
- テキスト1冊 = 10,000〜20,000円
- 月額980円 = 「週1本の有益記事で元が取れる」感覚
- **Netflixより安い** → 心理的ハードルが低い
- 競合（EDUWARD Pressなど）= 月額1,100〜2,200円 → やや下で勝負

### 6. 記事投稿時のSEO/拡散チェックリスト

```
□ タイトルは40文字以内で検索キーワードを含むか？
  良い例: 「犬の急性下痢 ─ メトロニダゾールは不要？最新RCTまとめ」
  悪い例: 「下痢について考える」

□ 最初の3行に結論が書いてあるか？
  → NoteのプレビューはOGP+最初の3行で決まる

□ ハッシュタグは適切か？（5〜10個）

  + テーマ固有: #急性下痢 #メトロニダゾール #プロバイオティクス

□ アイキャッチ画像は設定したか？
  → Xでシェアされた時のCTRが2〜3倍変わる

□ X告知ツイートは用意したか？
  → 記事公開と同時にXで投稿

□ マガジンに追加したか？

□ 過去記事からのリンクは張ったか？
  → 関連記事を相互リンクで回遊率UP
```

## ⚠️ 法的注意事項

- **免責事項**: 記事末尾に「臨床判断の参考資料です。治療方針は担当獣医師の判断に委ねられます」を必ず記載
- **著作権**: 論文の図表は引用の範囲で使用。全文転載はNG
- **薬機法**: 「この薬を使え」ではなく「○○が推奨されている」という書き方で
- **個人情報**: 症例紹介では特定できない情報に限定
