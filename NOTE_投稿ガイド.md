# 📝 Note 投稿ガイド ─ VetEvidence

## 1. Note投稿の具体的手順

### 🖥️ PCからの投稿（推奨）

```
① note.com にログイン
② 画面右上の「投稿」ボタン → 「記事を書く」をクリック
③ 見出し画像を設定（1280×670px推奨）
   → Canvaで作成した画像をアップロード
   ※ 2025年12月でCanva直接連携は終了。事前にDLしてアップロード
④ タイトルを入力
⑤ 本文を入力（Markdown風のエディタ）
   - 見出し（H2/H3）
   - 太字
   - 引用
   - 画像挿入（段落の「＋」マーク → 画像）
   - リンク埋め込み
⑥ 右上の「公開設定」ボタンをクリック
⑦ 公開設定画面で以下を設定：
   ├─ ハッシュタグ（1〜3個、後述）
   ├─ マガジンへの追加（該当するものを選択）
   ├─ 無料/有料の設定
   │   └─ 有料の場合：有料ラインをドラッグで設定 + 価格入力
   ├─ 公開日時の設定（予約投稿可能）
   └─ その他（コメント許可、SNSシェア設定等）
⑧ 「投稿する」をクリック → 公開完了！
```

### 📱 スマホアプリからの投稿

```
① noteアプリを開く
② 画面下部中央の「つくる」アイコンをタップ
③ 「記事を書く」をタップ
④ タイトルと本文を入力
⑤ 右上「公開設定」をタップ
⑥ ハッシュタグ・マガジン・販売設定を行う
⑦ 「公開する」をタップ → 完了
```

> 💡 **推奨**: 長文記事はPCで作成し、スマホは校正・確認用に使う

## 2. 有料記事の設定方法

### 初回のみ必要な手続き
- 本人情報の登録（氏名・住所）
- 振込先口座の設定

### 有料ラインの設定
```
記事の本文中で「ここから有料」のラインを自由に設定可能

【無料で見せる部分】
├─ 結論（30秒でわかる要点）
├─ 治療フロー表
└─ 昔vs今の比較表

────── ここに有料ラインを設定 ──────

【有料部分】
├─ 詳細な解説・データ
├─ 飼い主説明スクリプト
├─ 臨床Tips（現場のコツ）
└─ 参照論文リスト
```

### 価格設定（Phase 2以降）
| タイプ | 価格帯 | 備考 |
|--------|--------|------|
| 単体記事 | ¥300〜500 | コーヒー1杯程度 |
| マガジン（月額） | ¥980 | 月4〜8記事が全て読める |
| マガジン（年額） | ¥9,800 | 2ヶ月分お得 |

## 4. ハッシュタグ戦略

### 基本ルール
- **1記事あたり1〜3個**（多すぎるとスパム判定）
- 記事内容と**100%一致**するものだけ
- ビッグタグ（競合多）+ ミドルタグを組み合わせる

### VetEvidence 標準ハッシュタグ

| ジャンル | 推奨タグ |
|---------|---------|
| 全記事共通 | `#獣医` `#獣医師` |
| 救急 | `#救急` `#動物病院` |
| 循環器 | `#犬の心臓病` `#僧帽弁閉鎖不全症` |
| 皮膚 | `#犬のアトピー` `#犬の皮膚病` |
| 腎泌尿器 | `#猫の腎臓病` `#CKD` |
| 輸液 | `#輸液` |
| 麻酔 | `#麻酔` `#動物看護師` |
| 猫専門 | `#猫` `#猫の病気` |
| 腫瘍 | `#犬のがん` `#リンパ腫` |

### 投稿例
```
タイトル: 【輸液の基本】晶質液 vs 膠質液 ─ 2024年AAHAガイドラインの要点
ハッシュタグ: #獣医  #輸液  #動物病院
```

## 5. 投稿前チェックリスト

毎回の投稿前にこれを確認：

```
□ タイトルは具体的で分かりやすいか（30文字以内推奨）
□ 見出し画像は設定したか（Canvaで作成、1280×670px）
□ 結論が最初の3行以内にあるか
□ 読了時間の目安を冒頭に記載したか
□ 有料ラインの位置は適切か（無料部分だけでも価値があるか）
□ ハッシュタグは1〜3個、記事内容と一致しているか
□ マガジンに追加したか
□ 誤字脱字チェックは完了したか
□ 公開日時は正しいか（月or木の朝7時）
□ X告知用の文面は準備したか（140字以内の要約＋URL）
```

## 6. X（Twitter）連携の投稿フロー

```
📅 公開日（月or木）の朝
├─ 07:00 Note記事を公開（予約投稿推奨）
├─ 07:05 X で告知ツイート（記事リンク＋要約）
├─ 12:00 X でリマインドツイート（異なる切り口で）
└─ 19:00 X で関連豆知識or補足ツイート

📅 記事がない日（火・水・金・土・日）
├─ 「昔 vs 今」シリーズ
├─ 症例クイズ
├─ 論文ピックアップ
├─ コミュニティ交流（リプ・引用RT）
└─ 曜日別テーマ投稿
```
