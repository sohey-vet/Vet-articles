# 記事17: 猫の動脈血栓塞栓症（ATE）─ 初期対応と予後

## タイトル
🚨 猫の動脈血栓塞栓症（ATE）─ 初期対応と予後

## サブタイトル
「突然、後ろ足が動かなくなり泣き叫ぶ猫」── ATEの診断（The 5 Ps）、最優先すべきオピオイド鎮痛、ヘパリンを用いた急性期抗血栓療法、そして厳しい予後判定。

## 結論
猫のATEは**肥大型心筋症（HCM）を基礎疾患に持つことが多く**、左心耳内に形成された血栓が大動脈三叉部（鞍状血栓）に塞栓する。来院時は極度の疼痛を伴い、診断は**The 5 Ps**（Pain, Paresis/Paralysis, Pulselessness, Pallor, Poikilothermy）で行う。治療の最優先は**強力な鎮痛**であり、**Fentanyl Bolus 2-5 mcg/kg IV → CRI 2-5 mcg/kg/hr** が推奨される。Buprenorphine 0.01-0.02 mg/kg IV/IM q6-8h は重度疼痛には不十分な場合がある。急性期の抗血栓療法は**未分画ヘパリン（UFH）初回 250-300 IU/kg IV → 維持 150-250 IU/kg SC q6-8h**（APTTを1.5-2.0倍に調整）。低分子ヘパリン（LMWH）ではDalteparin 100-150 IU/kg SC q12-24hまたはEnoxaparin 1 mg/kg SC q12hも代替となる。慢性期・再発予防の第一選択は**Clopidogrel 18.75 mg/cat PO q24h**（FATCAT studyでAspirinより優位性が証明）。**Rivaroxaban 2.5 mg/cat PO q24h**（0.5-1 mg/kg）とのDual therapyは出血リスクが若干増えるが生存期間延長の報告がある。全体の生存退院率は**30-40%**、再発率は**25-50%**と厳しく、特に直腸温 < 37.0℃、徐脈、両後肢完全麻痺、複数部位の塞栓は悪い予後因子である。

## 診断 ─ The 5 Ps: 現場で見逃さない5つの徴候

ATEの診断は、高度な画像検査を待つまでもなく、身体検査だけでほぼ確定できる。以下の**5つのP**を系統的に確認する。

| 徴候 | 英語 | 詳細 |
| :--- | :--- | :--- |
| **1. Pain（疼痛）** | Pain | 激しい痛みにより泣き叫ぶ。パニック状態になることも多い |
| **2. Paresis/Paralysis（不全麻痺/麻痺）** | Paresis/Paralysis | 後肢の脱力〜完全な運動不全。尾のトーヌスも消失 |
| **3. Pulselessness（脈拍消失）** | Pulselessness | **大腿動脈の触知不可**。ドップラーで血流音を確認 |
| **4. Pallor（蒼白）** | Pallor | 肉球・爪床が青白〜紫色。健側肢と比較して判断 |
| **5. Poikilothermy（変温/冷感）** | Poikilothermy | 患肢が健側に比べ明らかに冷たい |

## 昔の常識 vs 今のエビデンス

| ❌ 旧来 | ✅ 最新 |
| :--- | :--- |
| とりあえずレントゲンを撮る | **鎮痛が最優先**。極度の疼痛とパニックで突然死するリスクがある |
| Aspirinで再発予防 | FATCAT studyにより**Clopidogrel 18.75 mg/catの優位性**が証明。Aspirinは推奨されない |
| 積極的に血栓を「溶かす」（tPA等） | 血栓溶解薬はカリウムや乳酸の急激な全身還流（**再灌流障害**）により致死的になりやすい |
| BuprenorphineでOKと判断 | ATEの疼痛は極めて強度。**Buprenorphineでは不十分**なことが多く、Fentanyl CRIが推奨される |

## 詳細解説

### 1. 鎮痛 ─ 最優先事項

ATEによる疼痛は獣医救急で遭遇する中でも最も激しいものの一つであり、鎮痛が不十分な場合、ストレスと疼痛そのものによる心停止のリスクがある。**検査や処置の前に、まず痛みを取ること**。

| 薬剤 | 投与経路 | 用量 | 備考 |
| :--- | :--- | :--- | :--- |
| **Fentanyl（推奨）** | IV → CRI | Bolus: **2-5 mcg/kg IV** → CRI: **2-5 mcg/kg/hr** | 最も迅速で調整しやすい。CRIで持続的な鎮痛を確保 |
| **Methadone** | IV or IM | **0.1-0.2 mg/kg q4-6h** | Fentanyl CRIが使えない場合の代替 |
| **Hydromorphone** | IV or IM | **0.05-0.1 mg/kg** | Methadoneの代替 |
| **Buprenorphine** | IV or IM | **0.01-0.02 mg/kg q6-8h** | **ATEの重度疼痛には不十分な場合あり**。部分アゴニストであり天井効果がある |

**💡 臨床Tips ─ なぜBuprenorphineでは不十分なのか**
*   Buprenorphineはμ受容体の**部分アゴニスト**であり、「痛みが取りきれない天井」がある。ATE特有の虚血性+神経障害性の激痛に対してはフルアゴニストであるFentanylやMethadoneが必要。
*   Fentanyl CRIのメリットは、鎮痛レベルを分単位で微調整でき、呼吸抑制が出た場合にも迅速に減量・中止できる点にある。

### 2. 抗血栓療法 ─ 急性期

**血栓溶解療法（tPA等）は推奨されない。** 溶解した血栓に含まれる高濃度のカリウムと乳酸が全身循環に一気に還流し、致死的な**再灌流障害**（超急性の高カリウム血症・不整脈・代謝性アシドーシス）を引き起こすリスクが非常に高いためである。

#### 未分画ヘパリン (UFH)
| 項目 | プロトコル |
| :--- | :--- |
| **初回投与** | **250-300 IU/kg IV** |
| **維持投与** | **150-250 IU/kg SC q6-8h** |
| **モニタリング** | **APTT を 1.5-2.0倍** に維持 |

#### 低分子ヘパリン (LMWH) ─ 代替
| 薬剤 | 用量 | 備考 |
| :--- | :--- | :--- |
| **Dalteparin** | **100-150 IU/kg SC q12-24h** | 高価 |
| **Enoxaparin** | **1 mg/kg SC q12h** | ─ |

### 3. 再発予防 ─ 慢性期

ATEは再発率が**25-50%**と非常に高い疾患である。退院時には長期的な抗血栓予防を必ず開始する。

| 薬剤 | 用量 | エビデンス |
| :--- | :--- | :--- |
| **Clopidogrel（第一選択）** | **18.75 mg/cat PO q24h**（75mg錠の1/4錠） | FATCAT studyで**Aspirinより優位性が証明**された |
| **Rivaroxaban（オプション）** | **2.5 mg/cat PO q24h**（0.5-1 mg/kg） | Clopidogrelとの**Dual therapy**が可能。出血リスクがわずかに増加するが生存期間延長の報告あり |

**⚠️ Aspirinはもう使わない**
*   FATCAT study（Hogan et al., JVIM 2015）により、Clopidogrelは再発率・生存率ともにAspirinより有意に優れていることが示された。現在のCURATIVE Guidelinesでも**Clopidogrelが第一選択**とされている。

### 4. 予後判定 ─ 厳しい現実と家族への伝え方

生存退院率は全体で**30-40%**（安楽死を含む母集団）であり、治療を開始した症例に限定するともう少し高いが、依然として楽観できる数字ではない。

#### 悪い予後因子
*   直腸温が **< 37.0℃**（低体温）
*   **徐脈**の存在（高カリウム血症やショックの示唆）
*   **両後肢**の完全な運動機能喪失（鞍状血栓）
*   **複数部位**の同時塞栓（例：右前肢にも症状）

**💡 臨床Tips ─ 飼い主さんへの伝え方**
*   「この病気は、仮に退院できたとしてもまた起こる可能性（25-50%）がある病気です。」
*   「治療費と通院の負担、そして再発のリスクを踏まえたうえで、〇〇ちゃんに一番良い選択を一緒に考えましょう。」
*   苦痛がコントロールできない、または四肢の壊死が進行する場合、安楽死は**医学的にも倫理的にも妥当な選択肢**であることを穏やかに、しかし明確に伝える。

## 飼い主への説明ガイド

### 「足が動かなくなって叫んでいたのですが…」
猫ちゃんの足に行く血管が急に詰まってしまう病気で、**動脈血栓塞栓症**といいます。とても強い痛みが出るので、まず最初に痛み止めの注射をしました。この病気は多くの場合、心臓の病気が原因で起こります。

### 「治るのでしょうか？」
正直にお伝えすると、この病気は非常に重い病気で、**退院できる確率は30〜40%**です。また退院できたとしても、**再発する可能性が25〜50%**あります。ただし、適切なお薬（血をサラサラにするお薬）を飲み続けることで、再発リスクを減らすことはできます。
