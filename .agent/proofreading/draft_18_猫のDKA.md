# 記事18: 猫の糖尿病性ケトアシドーシス（DKA）─ 緊急管理

## タイトル
🚨 猫の糖尿病性ケトアシドーシス（DKA）─ 緊急管理

## サブタイトル
「元気食欲が全くない」糖尿病猫 ── インスリン欠乏と極度の脱水。初期の輸液・カリウム補正から、レギュラーインスリン（RI）の持続点滴。早すぎるインスリンは致死的低カリウム血症を招く。

## 結論
DKAはインスリン欠乏（またはインスリン拮抗状態）により肝臓でのケトン体（β-ヒドロキシ酪酸、アセト酢酸）産生が亢進し、重度の代謝性アシドーシスと脱水をきたす内分泌の緊急疾患である。猫のDKA治療の三大柱は**「1. 脱水補正（輸液）」「2. 電解質（特にKとP）の補正」「3. インスリン投与」**であり、この順番を守ることが極めて重要である。来院時は**0.9% NaCl等の等張液**で循環血液量を回復させ、KClを添加して**カリウムの補正を先行**させる。来院時のK値が「正常」でも、アシドーシスによる細胞外シフトで見かけ上高く見えているだけであり、**全身のカリウム総量は著しく枯渇**している。インスリン投与は**カリウム値が安定し（> 3.0-3.5 mEq/L）、循環血液量が改善した後（通常2-6時間後）**に開始する。レギュラーインスリン（RI）はCRI **0.05-0.1 U/kg/hr**（猫では低用量から開始）またはIM投与（初回0.2 U/kg IM → 1-2h毎に0.1 U/kg IM）。血糖値が**250 mg/dL以下**に低下したら、ケトン消失のため**インスリンは継続しつつ、輸液に2.5-5%ブドウ糖（Dextrose）を添加**する。低リン血症（< 1.5 mg/dL）は**溶血性貧血**を引き起こすため、リン酸カリウム（K₂HPO₄）での補充を検討する。重炭酸塩（Bicarbonate）投与は通常不要で、pH < 7.0の極端な場合のみ慎重に検討する。

> ⚠️ **注意**: DKA専用のファクトシートは現時点で存在しない。上記の数値は`factsheet_diabetes_mellitus.md`および標準的な獣医救急テキスト（Silverstein & Hopper: Small Animal Critical Care Medicine; Plunkett: Emergency Procedures for the Small Animal Veterinarian）に基づく。工程③の検証時にGeminiによるクロスチェックが必須である。

## DKA治療のタイムライン

| 時間経過 | アクション | 目標・注意点 |
| :--- | :--- | :--- |
| **来院直後〜2-6時間** | **初期輸液と電解質補正** | 0.9% NaClで脱水補正。KCl添加開始。**まだインスリンは開始しない** |
| **K値安定後（2-6時間後）** | **RI投与開始** | CRI 0.05-0.1 U/kg/hr または IM (0.2 U/kg → 0.1 U/kg q1-2h)。血糖値を50-75 mg/dL/hrペースで緩やかに下げる |
| **血糖値 < 250 mg/dL到達** | **ブドウ糖の添加** | 輸液に2.5%または5%のDextroseを添加。**RIはケトン消失まで止めない** |
| **ケトン消失・食欲回復** | **皮下インスリンへ移行** | Glargine等の長時間作用型インスリンへ切り替え |

## 詳細解説

### 1. 電解質管理 ─ DKA治療の生命線

#### カリウム（K）の落とし穴
*   来院時の血中K濃度が「正常」であっても、**アシドーシスと高浸透圧**によって細胞内からKが細胞外へ押し出されているだけで、全身のカリウム総量は著しく枯渇している。
*   インスリン投与によってKが細胞内に戻ると、急激な**低K血症**（筋力低下、頭部下垂〔ventroflexion: 猫に特徴的〕、呼吸筋麻痺）が発生する。
*   輸液開始時点から**積極的なKCl添加**が必要。一般的な添加量の目安:

| 血清K (mEq/L) | KCl添加量 (mEq/L) | 最大投与速度 |
| :--- | :--- | :--- |
| > 3.5 | 20 | 0.5 mEq/kg/hr（急速ボーラス等での制限超えに注意） |
| 3.0 - 3.5 | 40 | 0.5 mEq/kg/hr |
| 2.5 - 3.0 | 60 | 0.5 mEq/kg/hr |
| < 2.5 | 80 | 0.5 mEq/kg/hr（心電図モニタリング） |

> ⚠️ **注意**: DKA専用ファクトシートに記載なし。上記のKCl添加スライディングスケールは標準的な獣医救急テキストに基づく一般的な参考値であり、症例ごとの調整が必要。

#### リン（P）の減少
*   インスリン投与によりリンも細胞内に移動する。
*   血中リン濃度が **< 1.5 mg/dL** になると、赤血球内のATPと2,3-DPGが枯渇し、**重度の溶血性貧血**を引き起こす。
*   輸液のKCl補充の一部を**リン酸カリウム（K₂HPO₄）**に置き換えることで、KとPを同時に補充できる（ただしCa沈着のリスクがあるため全量をリン酸カリウムにしない）。

### 2. レギュラーインスリン（RI）投与プロトコル

#### CRI法（推奨）
*   **方法**: 0.9% NaCl 250mLバッグにRI 50Uを添加（= 0.2 U/mL）し、シリンジポンプ等で**独立したラインから投与**。
*   **開始用量**: CRI **0.05-0.1 U/kg/hr**（猫では0.05 U/kg/hrから慎重に開始することが多い）。
*   **メリット**: 血糖値に応じた分単位の微調整が容易。低血糖時には直ちにレートダウンまたは一時中断が可能。
*   **注意**: RIはプラスチック製チューブに吸着するため、**最初の50mL程度**はプライミングとしてラインに流して捨てる。

#### IM法（シリンジポンプがない場合）
*   **方法**: 初期に **0.2 U/kg IM**。その後1-2時間ごとに **0.1 U/kg IM**。
*   **注意**: 脱水が解消されるまではSC（皮下）投与は吸収が不安定なので避ける。

### 3. 血糖値が下がってもインスリンを止めてはいけない理由

DKAの治療目標は血糖値の正常化ではなく、**ケトン体の消失**である。インスリンを止めると肝臓でのケトン産生が再開するため、血糖値が250 mg/dLを下回った時点で**輸液にDextroseを添加**し、インスリンは少量でも継続する。

*   血糖値 < 250 mg/dL → 輸液に **2.5% Dextrose** 添加
*   血糖値 < 150 mg/dL → 輸液に **5% Dextrose** 添加
*   ケトン陰性＋食欲回復＋酸塩基平衡正常化 → **皮下インスリン（Glargine等）へ移行**

### 4. 併発疾患の検索 ─ DKAを引き起こした「引き金」

DKAは「何かがインスリンの効きを悪くした結果」起こることが多い。以下の併発疾患を積極的に検索する:
*   感染症（UTI、皮膚感染、肺炎）
*   膵炎（猫に極めて多い ── fPLIの確認）
*   腎臓病
*   腫瘍（副腎皮質機能亢進症は犬に多いが、猫では稀）

## 飼い主への説明ガイド

### 「うちの子はもう糖尿病のお薬を飲んでいたのに…」
糖尿病の猫ちゃんは、他の病気（膀胱炎や膵炎など）がきっかけでインスリンの効きが悪くなり、**体の中に毒素（ケトン体）が溜まる**ことがあります。これが「ケトアシドーシス」で、緊急の入院治療が必要です。

### 「治りますか？」
適切な治療を行えば、多くの猫ちゃんが数日〜1週間ほどで回復します。ただし最初の24-48時間は非常に不安定で、点滴やインスリンの量をこまめに調整しなければなりません。退院後は、DKAを引き起こした原因（例えば膀胱炎など）の治療と、インスリンの用量の再調整が必要になります。
