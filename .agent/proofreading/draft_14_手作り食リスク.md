# 記事14: 手作り食・生食のリスクとエビデンス

## タイトル
🍽️ 手作り食・生食のリスクとエビデンス

## サブタイトル
「自然な食事が一番」は本当か ── 市販生食の20-50%からサルモネラが検出される事実。カルシウム欠乏で骨折する子犬。科学的根拠に基づくリスク評価。

## 結論
生食（Raw Meat Based Diets, RMBD）には公衆衛生リスクと栄養学的リスクの双方が存在する。市販RMBDの**20-50%からサルモネラ属菌が検出**され、RMBDを摂取した犬猫は無症状でも糞便から病原菌を排泄し、同居する免疫不全者（幼児・高齢者・妊婦・化学療法中の人）への感染源となり得る。栄養面では、正肉のみの給与はCa/P比が低く**栄養性二次性上皮小体機能亢進症**（骨折・くる病）を引き起こし、生魚のチアミナーゼにより**チアミン（ビタミンB1）欠乏**（猫に感受性高い）、レバーの多給で**ビタミンA過剰**（骨変形）が起こる。「毛艶改善」「免疫力向上」などの主張に対する科学的根拠（査読付き論文）は**存在しない**。WSAVA・AVMA・CDCは犬猫への生肉給与を推奨しておらず、手作り食を希望する場合は**獣医栄養学専門医（DACVN/ECVCN）のレシピ**に従うことが必須。

## 生食・手作り食のリスク一覧

| リスクカテゴリ | 具体的リスク | 結果 |
| :--- | :--- | :--- |
| **細菌汚染** | Salmonella, Listeria, Campylobacter, E. coli O157 | 動物は無症状でも人間に感染（Zoonosis） |
| **Ca欠乏** | 正肉のみでCa/P比が崩壊 | 栄養性二次性上皮小体機能亢進症（骨折） |
| **チアミン欠乏** | 生魚のチアミナーゼ | 神経症状（猫で特に感受性高い） |
| **ビタミンA過剰** | 生レバーの多給 | 骨変形（頚椎の骨棘形成） |
| **物理的障害** | 生骨・加熱骨 | 食道閉塞、胃穿孔、歯牙破折 |

## 昔の常識 vs 今のエビデンス

| ❌ 旧来 | ✅ 最新 |
| :--- | :--- |
| 「野生の食事に近い」から体に良い | 科学的根拠は**存在しない**。加熱による消化率向上と殺菌のメリットが大きい |
| ネットのレシピで栄養バランスは取れる | 書籍・ネットのレシピの多くは**栄養学的評価で不合格**。専門医の処方が必須 |
| 犬猫は生肉で食中毒にならない | 動物は無症状でも**糞便から病原菌を排泄**。人間（特に免疫不全者）への感染リスク |

## 詳細解説

### 公衆衛生リスク ─ 誰が本当に危険なのか
*   市販RMBD製品の**20-50%**からSalmonella spp.が検出（JAVMA研究）
*   犬猫は不顕性感染（無症状キャリア）→ 糞便中に病原菌を**排泄（Shedding）**
*   ハイリスク群: 幼児、高齢者、妊婦、化学療法中の患者 → 免疫力が低下しており重症化リスク

**⚠️ 獣医師として伝えるべきこと**
*   ご家族に小さなお子さんがいる場合、生食のリスクを**明確に説明する義務**がある
*   飼い主の選択を尊重しつつも、エビデンスに基づいた情報提供を怠らない

### 「それでも手作り食をしたい」場合の推奨事項
*   必ず**獣医栄養学専門医（DACVN/ECVCN）**の処方した完全なレシピに従うこと
*   サプリメント（Ca、ビタミンD、微量元素等）を正確に計量して添加
*   定期的な栄養状態の評価（血液検査含む）
*   生肉ではなく**加熱調理**を推奨（栄養学的に問題なく、感染リスクを大幅に低減）

**💡 日本の実情**
*   日本にはDACVN相当の獣医栄養学専門医が極めて少ない
*   BalanceIT.comなどのオンラインツールで栄養計算を行う方法もあるが、英語対応
*   現実的には**AAFCO基準を満たした総合栄養食**を基本とし、手作りはトッピング程度に留めるのが安全

## 飼い主への説明ガイド

### 「手作りの方が愛情がこもっていて良いのでは？」
お気持ちは素晴らしいと思います。ただ、犬や猫に必要な栄養素は**人間とはまったく違います**。例えばカルシウムが足りないと骨がもろくなりますし、猫に生の魚を多く与えるとビタミンB1が壊されて神経症状が出ることがあります。手作り食をされるなら、**獣医栄養学の専門家のレシピ**に従うことをお勧めします。
