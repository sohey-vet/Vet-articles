# 記事19: 犬の高浸透圧高血糖症候群（HHS）─ DKAより恐ろしい罠

## タイトル
🐶 犬の高浸透圧高血糖症候群（HHS）─ DKAより恐ろしい罠

## サブタイトル
著しい高血糖（>600mg/dL）、高浸透圧、しかしケトン陰性 ── 極度の脱水に適応した脳細胞。急速な輸液とインスリン投与は致死的な「脳浮腫」を引き起こす。DKAとは根底から違う超・徐緩なマネジメント。

## 結論
HHS（Hyperosmolar Hyperglycemic Syndrome / State）は、少量のインスリンが残存しているためケトン体産生は抑制されているが、**極端な高血糖（通常 > 600 mg/dL、しばしば > 800-1000 mg/dL）と重度の脱水、血清浸透圧の著増（> 350 mOsm/kg）**により、重篤な神経症状（昏睡、発作）を呈する内分泌の緊急疾患である。犬猫ともに稀だが、猫での報告が犬よりも多いとされる。脳細胞は高い細胞外浸透圧に対抗するために数日かけて**特発性浸透圧物質（idiogenic osmoles: タウリン、ミオイノシトール等のアミノ酸類）**を細胞内に蓄積し、細胞体積を維持するという防御機構を発動している。この状態で**急速な輸液や早期のインスリン投与**を行い細胞外浸透圧を急激に低下させると、特発性浸透圧物質が溜まった脳細胞内に水が一気に流入し、**致死的な脳浮腫（Cerebral edema）**を引き起こす。治療の絶対原則は**「超緩徐」**であり、インスリンの開始は**数時間〜半日以上待ち**、0.9% NaCl等の等張液だけで**36-48時間かけて**血清浸透圧と脱水を補正する。血清浸透圧の低下速度は **0.5-1.0 mOsm/kg/hr以下** に抑えることが推奨される。DKAとは治療プロトコルが根本的に異なるため混同は致命的である。

## DKA と HHS の鑑別 ─ 見た目は似ているが治療は真逆

| 項目 | DKA（ケトアシドーシス） | HHS（高浸透圧高血糖） |
| :--- | :--- | :--- |
| **血糖値** | 通常 300-500 mg/dL 程度 | **極度（> 600-1000 mg/dL 以上）** |
| **ケトン体** | **陽性（強）** | **陰性または弱陽性** |
| **血清浸透圧** | やや高値（< 350 mOsm/kg） | **著増（> 350 mOsm/kg）** |
| **アシドーシス** | 中等度〜重度 | なし〜軽度 |
| **神経症状** | 虚弱、嘔吐（アシドーシスによる） | **深い昏睡、発作、脳症**（高浸透圧による） |
| **輸液ペース** | 12-24時間で脱水補正 | **36-48時間（超緩徐）** |
| **インスリン開始タイミング** | 輸液開始2-6時間後 | **十分な脱水補正後（数時間〜半日以上待つ）** |
| **最大の脅威** | 低K血症、代謝性アシドーシス | **脳浮腫** |

## 昔の常識 vs 今のエビデンス

| ❌ 絶対やってはいけない「罠」 | ✅ 正しいアプローチ |
| :--- | :--- |
| 低張液（0.45% NaCl等）を大量に投与 | **0.9% NaCl（等張液）から開始**。低張液は浸透圧の急激な低下を招き脳浮腫の直接原因となる |
| 来院直後からインスリンで血糖値をガンガン下げる | **インスリンは最初は打たない**。まずは輸液だけで数時間かけて浸透圧利尿によるわずかな血糖低下を見守る |
| DKAと同じプロトコルで血糖値を50-75 mg/dL/hrペースで下げる | HHSでは**それよりも遥かに遅いペース**が必要。血清浸透圧の変化を0.5-1.0 mOsm/kg/hr以下に抑える |
| 有効浸透圧の計算をおろそかにする | **必ず「有効」血清浸透圧を計算**し、輸液速度の指標にする。計算式: 2×Na + (Glu/18) |

## 詳細解説

### 1. なぜ脳浮腫が起きるのか ─ 病態生理の理解が治療の鍵

#### 高浸透圧への脳の「適応」
1.  著しい高血糖により、細胞外液の浸透圧が慢性的に上昇する。
2.  通常であれば、浸透圧較差によって脳細胞から水分が外に引き出され、脳細胞は縮小する（脱水される）。
3.  しかし脳細胞は数日かけてこの状態に「適応」し、細胞内に**特発性浸透圧物質（idiogenic osmoles）**──タウリン、グルタミン酸、ミオイノシトールなどのアミノ酸・有機溶質──を産生・蓄積して、細胞内浸透圧を上げ、体積を保つ。

#### 急激な補正 → 致死的な水の逆流入
4.  この「バランスが取れた」状態の脳に対して、獣医師が急激にインスリンや低張液を投与し細胞外の血糖=浸透圧を下げると、**特発性浸透圧物質が溜まったままの細胞内に水が一気に逆流入**する。
5.  脳は頭蓋骨という閉鎖空間にあるため、膨張の逃げ場がなく**致死的な脳浮腫**を引き起こす。

**💡 臨床Tips ─ 一言で表すなら**
「脳は高い浸透圧に数日かけて適応した。我々がそれを数時間で元に戻そうとすれば、脳は壊れる。」

### 2. HHS治療プロトコル ─ 超緩徐がすべて

#### Step 1: 輸液（等張液で超緩徐に）
*   **0.9% NaCl**で開始する。**低張液（0.45% NaCl）は使わない。**
*   脱水量の補充は**最低36-48時間**をかけて行う。
*   ショック状態でない限り、ボーラス投与は避ける。ボーラスが必要な場合も少量（5-10 mL/kg）を短時間で慎重に投与する。
*   輸液開始後、細胞外液量の増加によりGFR（糸球体濾過量）が改善し、**浸透圧利尿によってグルコースが尿から排泄**されるだけで、血糖値は安全なペースで緩やかに低下し始める。

#### Step 2: インスリンの「遅延」開始
*   輸液開始後、水和状態が改善し、血清浸透圧の低下が安定するまで**インスリンの投与は数時間〜半日以上待つ**。
*   インスリンを開始する場合も、**DKAより低い用量**で慎重に投与（CRI 0.025-0.05 U/kg/hr など）。
*   目標は血清浸透圧の変化が **0.5-1.0 mOsm/kg/hr以下** になること。

#### Step 3: 脳浮腫の徴候監視
*   治療中に突然の以下の症状が出現した場合、脳浮腫を強く疑う:
    *   意識レベルの急激な低下
    *   発作
    *   徐脈
    *   瞳孔不同
*   脳浮腫が疑われた場合: **20%マンニトール 0.5-1.0 g/kg IV を15-20分かけて投与**

### 3. 血清浸透圧の計算 ─ 必ず計算する

血清浸透圧は以下の式で推定できる:

```
有効血清浸透圧 (Effective Osmolality) = 2 × Na (mEq/L) + Glucose (mg/dL) / 18
```

（全血清浸透圧には BUN/2.8 も加えるが、BUNは細胞膜を自由に通過するため「有効」浸透圧には含めないのが標準）

*   **正常**: 290-310 mOsm/kg
*   **HHSの典型**: > 350 mOsm/kg
*   **治療目標**: 浸透圧の低下を **< 0.5-1.0 mOsm/kg/hr** に保つ

### 4. 予後

*   HHSは発生頻度こそDKAより低いものの、死亡率は**40-70%**（あるいはそれ以上）と非常に高い。
*   死因の多くは脳浮腫のほか、併発疾患（腎不全、敗血症等）による多臓器不全。
*   早期認識と「DKAとは異なるプロトコル」の適用が生存率向上の鍵。

## 飼い主への説明ガイド

### 「血糖がすごく高いのに、すぐインスリンを打ってくれないのですか？」
お気持ちはよく分かります。ただ、この状態でいきなりインスリンを打って血糖値を急に下げてしまうと、**脳が腫れてしまう（脳浮腫）**危険性が非常に高いのです。脳は数日かけてこの「高い状態」に適応しているので、**元に戻すにも同じくらいゆっくり、48時間ほどかけて慎重に**治療する必要があります。

### 「助かりますか？」
この病気は非常に重く、死亡率も高い病気です。ただし、慎重な輸液管理でゆっくりと状態を安定させることで回復する子もいます。最初の24-48時間の**慎重な管理が勝負**です。
