# 記事13: てんかんの長期管理 ─ 抗てんかん薬の選択

## タイトル
🧠 てんかんの長期管理 ─ 抗てんかん薬の選択

## サブタイトル
「発作は止まったけど、薬は一生飲むの？」── Phenobarbitalの血中濃度モニタリング、KBrの猫への禁忌、Levetiracetamの"Honeymoon effect"。正しい薬剤選択と飼い主への説明術。

## 結論
抗てんかん薬（AED）の開始基準は、**6ヶ月間に2回以上**の発作、重積/群発発作の既往、発作後徴候が重度/24時間以上持続、または構造的てんかんの診断。犬猫の第一選択薬は**Phenobarbital 2.5-3.0 mg/kg PO q12h**（目標血中濃度 **15-35 mcg/mL**、開始2-3週後に測定）。**臭化カリウム（KBr）20-40 mg/kg PO q24h**は犬の第一選択/追加薬だが、**猫では好酸球性肺炎リスクのため禁忌**。追加薬として**Levetiracetam 20 mg/kg PO q8h**は肝代謝を受けず安全性が高いが、"Honeymoon effect"（初期に効いて後に耐性がつく）に注意し、群発時のPulse therapyに活用する。**Zonisamide 5-10 mg/kg PO q12h**は日本で広く使われる追加薬。犬のImepitoin 10-30 mg/kg PO q12hは特発性てんかんに安全性が高いが群発/重積には不向き。

## 抗てんかん薬の比較

| 薬剤 | 用量 | 犬 | 猫 | 主な副作用 |
| :--- | :--- | :--- | :--- | :--- |
| **Phenobarbital** | 2.5-3.0 mg/kg PO q12h | ✅ 第一選択 | ✅ 第一選択 | PU/PD/PP、鎮静、肝毒性 |
| **KBr** | 20-40 mg/kg PO q24h | ✅ 第一選択/追加 | **❌ 禁忌** | PU/PD/PP、膵炎 |
| **Levetiracetam** | 20 mg/kg PO q8h | ✅ 追加薬 | ✅ 追加薬 | 少ない（Honeymoon注意） |
| **Zonisamide** | 5-10 mg/kg PO q12h | ✅ 追加薬 | ✅ 追加薬 | 肝毒性（稀）、KCS |
| **Imepitoin** | 10-30 mg/kg PO q12h | ✅ 第一選択 | ─ | 少ない。群発に不向き |

*PU/PD/PP = 多飲多尿多食。KCS = 乾性角結膜炎。*

## 昔の常識 vs 今のエビデンス

| ❌ 旧来 | ✅ 最新 |
| :--- | :--- |
| KBrは犬猫共通で使える | **猫ではKBr禁忌**（好酸球性肺炎のリスク） |
| Levetiracetamは単剤で長期管理できる | **Honeymoon effect**があるため、単剤維持よりも群発時のPulse therapyや追加薬向き |

## 詳細解説

### 治療開始基準 ─ いつ薬を始めるか
*   **6ヶ月間に2回以上**の発作
*   重積発作（Status Epilepticus）または群発発作（Cluster Seizures）の既往
*   発作後の徴候が**重度（攻撃性、長時間など）**または24時間以上続く
*   構造的てんかん（脳腫瘍、脳炎など）の診断

**💡 臨床Tips**
*   「年に1-2回の発作」でも飼い主が管理困難を訴えるなら、早期介入を検討してよい
*   発作日記（日時・持続時間・様子）を飼い主につけてもらうことが、投薬調整の基礎データになる

### Phenobarbitalの血中濃度モニタリング

| 項目 | 推奨 |
| :--- | :--- |
| **目標血中濃度** | **15 - 35 mcg/mL** |
| **初回測定** | 開始2-3週後（定常状態到達後） |
| **定期測定** | 6ヶ月ごと（肝酵素＋胆汁酸と同時） |
| **採血タイミング** | トラフ値（次回投薬直前） |

**⚠️ 肝毒性の管理**
*   ALP上昇は酵素誘導によるもので通常問題ないが、**ALTの上昇**は肝実質障害を示唆
*   血中濃度 > 35 mcg/mL で肝毒性リスク上昇 → 減量または薬剤変更
*   定期的に**胆汁酸測定**で肝機能を評価

### 臭化カリウム（KBr）─ 猫禁忌の理由と犬での運用
*   **猫では好酸球性肺炎（気管支炎）**を発症するリスクがあり**禁忌**
*   犬の用量: **20-40 mg/kg PO q24h**（半減期 約24日 ── 非常に長い）
*   急ぐ場合は負荷投与: 400-600 mg/kg を数回に分けて数日で投与（胃腸障害に注意）
*   食事の塩分濃度で血中Br-が変動する（食事の塩分↑ → Br排泄↑ → 血中Br↓）。フード変更時に注意

## 飼い主への説明ガイド

### 「薬は一生飲まないといけないの？」
多くの場合、はい。てんかんのお薬は**「発作を予防する」**ためのもので、病気を治すものではありません。お薬を急にやめると、かえって重い発作が起きることがあります。量を調整しながら、うまく付き合っていきましょう。

### 「太ってきたし、水もすごく飲むけど…」
Phenobarbitalには**食欲が増す・お水をたくさん飲む**という副作用があります。これは最初の数週間で特に目立ちます。食事量を発作前と同じ量に管理してあげてください。定期的に血液検査で肝臓への影響がないか確認しますので、一緒に見守りましょう。
