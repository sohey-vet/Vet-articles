# 獣医学記事 タグ付けルールと運用ガイドライン

本スキルは、VetEvidenceプロジェクトにおける「記事のタグ付け（ジャンルおよびサブタグ）」に関する厳格なルールと、2026年3月時点でのタグ整理状況をまとめたものです。
新規記事の作成時や、既存タグの見直しの際には必ず本ルールを参照してください。

## 1. タグ付けの基本原則

- **ジャンルタグ（メインタグ）は必ず1つ以上指定する**
  サイトのサイドバー（ジャンルフィルター）と連動するため、以下の「許可されたジャンルタグ」の中から最も適切なものを必ず先頭（メイン）に設定すること。
- **表記揺れの防止**
  - `犬`、`犬猫共通`などの動物種タグは使用しない。**`猫`のみ許可**（猫特有の疾患や注意点がある場合のみ）。
  - `栄養学`は禁止し、`栄養`に統一。
  - `泌尿器`は禁止し、`腎泌尿器`に統一。
  - `免疫疾患`は禁止し、`免疫`に統一。
  - `消化器・肝臓`は禁止し、`消化器`（または`肝臓`）とする。
- **無駄なタグの排除**
  一般的すぎる用語（例：`診断`, `基礎`, `基本`, `ガイドライン`）、学会名・試験名（`AAHA`, `ACVIM`, `IRIS`, `EPIC試験`など）、ターゲット層（`一般病院向け`）、詳細すぎる・専門的すぎる用語（`薬理`, `PDH`, `MMVD`など）はタグとして抽出・付与しない。

## 3. サブタグの選定基準

ジャンル以外のタグ（サブタグ）は、以下の基準を満たすもののみ付与します（最大でも総数4〜5個程度まで）。

- **疾患名・症状名** (例: `中毒`, `ショック`, `肺水腫`, `膀胱炎`, `リンパ腫`, `FIP`など)
- **手技・治療法** (例: `外科`, `心電図`, `化学療法`, `食事療法`, `CPR`など)
- **特殊なカテゴリー** (例: `抗菌薬`, `産科`, `電解質`, `ステロイド`など)

## 3-1. 新規サブタグの採用基準（2026年3月確定ルール）

- **新しいサブタグ（ジャンルタグ以外）を採用するには、現時点または将来の投稿予定記事において、2本以上の記事で使用される見込みがあること**が必要。
- 1記事にしか使われない概念/用語はサブタグとして付与しない。
- ユーザーに候補を提示し確認を取ってから採用すること。

## 4. 記事作成ワークフローにおける「確認プロセス」

新規記事を作成する際（`medical-article.md` 工程④：整形時）には、**必ず以下のプロセスを踏むこと**。

1. **タグ案の生成**: 記事のテーマに基づき、本スキルのルールに則った「ジャンルタグ（1つ以上）」と「サブタグ（1〜3つ程度）」を起案する。
2. **ユーザーへの提示**: 最終ファイル（Markdown/HTML）を生成する**前**に、必ずユーザーに対してタグ案を提示する。
   - 例：「ジャンルは『救急』、サブタグとして『中毒』『電解質』を付与する予定ですが、よろしいですか？」
3. **合意後の出力**: ユーザーの承認を得た後、初めてMarkdownおよびHTMLファイルにタグを書き込み、保存・出力する。

## 5. 過去の整理履歴 (2026年3月実施)

- 全85記事のタグをスクリプトで一括パージし、シンプルかつ検索性の高いタグ体系へと移行した。
- 削除された主なタグ群：`犬`, `犬猫共通`, `ACVIM`, `AAHA`, `IRIS`, `ガイドライン`, `基本`, `鑑別診断`, `薬理` など多数。
- 統合・リネーム：`栄養学`→`栄養`、`泌尿器`→`腎泌尿器`、`免疫疾患`→`免疫`、`消化器・肝臓`→`消化器`。
- 新設された主なサブタグ：`ショック`, `中毒`, `産科`, `電解質`。
