# 📮 Note投稿の手順と規約

## 1. 記事変換（HTML → Note用）

```bash
python scripts/format_articles_for_note.py --file topics/カテゴリ/記事名.html
```

出力先: `topics_note/カテゴリ/記事名/` に変換されたファイルが生成される。

## 2. 自動投稿

```bash
python scripts/note_auto_post.py --file "topics_note/カテゴリ/記事名/記事名.md"
```

### オプション
- `--scheduled "YYYY-MM-DDTHH:MM"` — 予約投稿
- `--tags "タグ1,タグ2"` — ハッシュタグ指定

## 3. ハッシュタグルール

- **1記事あたり1〜3個**（多すぎるとスパム判定）
- 全記事共通: `#獣医` `#獣医師`
- ジャンル固有タグを1つ追加

| ジャンル | 推奨タグ |
|---------|---------|
| 救急 | `#救急` |
| 循環器 | `#犬の心臓病` |
| 皮膚 | `#犬の皮膚病` |
| 腎泌尿器 | `#猫の腎臓病` |
| 猫専門 | `#猫の病気` |

## 4. 投稿ログ

投稿結果は `note_post_log.json` に自動記録される。
構造:
```json
{
    "記事ファイル名": {
        "posted_at": "ISO8601日時",
        "status": "success|error",
        "url": "Note記事URL"
    }
}
```

## 5. 投稿スケジュール

- 月・水・金の週3回
- 昼12時公開（予約投稿推奨）
- `CONTENT_PLAN.md` のスケジュールに従う

## 6. 投稿前チェック

```
□ タイトルは具体的で30文字以内か
□ 結論が最初の3行以内にあるか
□ ハッシュタグは1〜3個か
□ 誤字脱字チェック済みか
□ X告知用テキストは準備したか
```
