# ✍️ Markdown記事 執筆規約

> **全ての記事はこの規約に従って執筆すること。**

## 1. MDファイルの必須構成

```markdown
# タイトル（臨床現場で検索しやすい名称）

## 結論 (Take-home Message)
（救急現場でまず確認すべき要点。2〜4文。）

## 本文
### セクション1のタイトル
内容...

### セクション2のタイトル
内容...

## 飼い主への説明ガイド
### Q. 「質問1」
回答1

### Q. 「質問2」
回答2

## 参照論文
1. 著者 et al. タイトル. 雑誌名 年;巻(号):ページ.
2. ...

---
tags: [タグ1, タグ2, タグ3]
description: 記事の概要（1文）
update: YYYY-MM-DD
```

## 2. 略語のルール

### 初出時は必ず日本語で展開
```
❌ CHFの治療として...
✅ CHF（うっ血性心不全）の治療として...
```

### 注釈不要リスト（獣医師なら全員知っている略語）
RR, HR, IV, IM, SC, SpO2, SID, BID, TID, SDMA, Cre, UPC, MRI, CT, NSAIDs, ACTH, IMHA, PCV, BUN, ALT, ALP, GGT, CRP

### 2回目以降はそのまま略語でOK

## 3. 数値の書き方

| 種類 | 書き方 | 例 |
|------|--------|-----|
| 投与量 | mg/kg | Prednisolone 2 mg/kg |
| 液量 | mL/kg | 0.5 mL/kg |
| CRI | µg/kg/min or mg/kg/h | Lidocaine 50 µg/kg/min |
| 濃度 | % | 50% Dextrose |
| 範囲 | 〜（全角チルダ） | 0.5〜1.0 mg/kg |
| 回数 | q12h, SID等 | PO q12h |

## 4. 推奨の強さの書き分け

| エビデンスレベル | 表現 |
|-----------------|------|
| RCT/ガイドライン | 「〜が推奨される」「〜すべきである」 |
| 複数の後ろ向き研究 | 「〜が有効とされる」 |
| 少数症例/専門家意見 | 「〜を検討する」「〜が報告されている」 |
| 論文に記載なし | 「論文内に記載なし」と明記 |

## 5. 日本の臨床実情への配慮

海外エビデンスと日本の実情にギャップがある場合、以下を記載:
- 日本で入手可能な薬剤・製品名
- 日本の犬種構成の違い（小型犬主体）
- 日本の動物病院の設備水準

## 6. 文体ルール

- 敬体（〜です/〜ます）ではなく、常体（〜である/〜する）で執筆
- ただし飼い主説明ガイドのみ敬体
- 箇条書きを多用し、パラグラフは短く
- 臨床的に重要なポイントは **太字** で強調
