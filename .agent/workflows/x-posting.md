# 🐦 X投稿のルールとフォーマット

## 1. 投稿テンプレート（4種類）

### ① 「昔の常識 vs 今」型（最もバズりやすい）
```
⚡ 昔の常識 vs 今のエビデンス

❌ 昔: [旧来の常識]
✅ 今: [最新のエビデンス]

📄 根拠: [著者] et al., [掲載誌], [年]

明日の診療から変えられます 💪

#獣医 #獣医師 #エビデンス #[テーマタグ]
```

### ② 「結論ファースト」型
```
🎯 [テーマ名] の結論

[1〜2文で結論]

詳しくは Note で👇
[リンク]

#獣医 #動物病院 #[テーマタグ]
```

### ③ 「数字で驚かせる」型
```
📊 知ってましたか？

[インパクトのある数字]

[補足説明1〜2文]

#獣医 #獣医師 #[テーマタグ]
```

### ④ 「症例クイズ」型
```
🐾 症例クイズ

[犬種/猫種]、[年齢]、[主訴]
[検査所見の要約]

あなたの次のアクションは？
↓ リプで教えてください

A) [選択肢A]
B) [選択肢B]
C) [選択肢C]

正解は今夜20時に🕗

#獣医クイズ #獣医師
```

## 2. ハッシュタグ

- **1投稿あたり5〜8個**
- 必須: `#獣医` `#獣医師` `#エビデンス`
- テーマ固有を2〜5個追加
- 多すぎるとスパム認定される

## 3. 最適投稿時間

| 時間帯 | ターゲット | おすすめ度 |
|--------|-----------|-----------|
| 7:00〜8:00 | 出勤前の獣医師 | ⭐ 最高 |
| 12:00〜13:00 | 昼休み | ◎ |
| 20:00〜22:00 | 仕事後 | ⭐ 最高 |

## 4. やってはいけないこと

- ❌ 宣伝ばかりの投稿（Note告知は週1〜2回）
- ❌ 一般飼い主向けの投稿（ターゲットがブレる）
- ❌ 診断・治療の断定（「〜すべき」→「〜が推奨されている」）
- ❌ 根拠のない主張（必ず論文を付ける）
- ❌ 他の獣医師の治療方針を批判
- ❌ 投稿を何日も空ける（最低でも週5）
