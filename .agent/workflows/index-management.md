# 🌐 index.html 管理ルール

> **新記事を追加したら、必ずindex.htmlにも登録すること。**

## 1. サイドバージャンルフィルタ（絶対厳守）

> ⚠️ **`data-tags` には、以下のジャンル名を最低1つ含めなければならない。含まれていないとサイドバーから絞り込めず記事が検索不能になる。**

| data-genre（サイドバー表示名） | 対応するdata-tagsの値 |
|------|------|
| 救急 | `救急` |
| 循環器 | `循環器` |
| 輸液 | `輸液` |
| 腎泌尿器 | `腎泌尿器` |
| 猫 | `猫` |
| 皮膚 | `皮膚` |
| 免疫 | `免疫` |
| 血液 | `血液` |
| 消化器・肝臓 | `消化器・肝臓` |
| 腫瘍 | `腫瘍` |
| 神経 | `神経` |
| 内分泌 | `内分泌` |
| 麻酔 | `麻酔` |
| 眼科 | `眼科` |
| 歯科 | `歯科` |
| その他 | `その他` |

**サイドバーに該当ジャンルがない記事は「その他」に分類すること。**

## 2. 記事カードの追加手順

新記事を `topics/` に追加したら、`index.html` の対応するカテゴリセクション（`<!-- ===== カテゴリ名 ===== -->`の後）に記事カードを追加する。

### 記事カードのHTML構造
```html
<a class="article-card" data-tags="ジャンル1,ジャンル2,追加タグ"
   href="topics/カテゴリ/ファイル名.html"><span
        class="card-icon">🏥</span>
    <h3>記事タイトル</h3>
    <p>記事の概要（1〜2文）</p>
    <div class="card-footer"><span class="tag tag--primary">タグ1</span><span
            class="tag tag--warning">タグ2</span><span
            class="card-papers">📄 参照論文X本</span></div>
</a>
```

## 3. 記事数の更新

記事の追加・削除後、サイドバーの `genre-count` とヘッダーの `visible-count` を更新する。

## 4. 注意事項

- ❌ **data-tagsにサイドバージャンル名を含めないのは絶対禁止**
- ❌ index.html のCSS/JSリンクのバージョンも `/version-control` に従う
- ❌ 記事カードの構造を勝手に変えない
- 新記事追加後は必ずブラウザで表示確認
