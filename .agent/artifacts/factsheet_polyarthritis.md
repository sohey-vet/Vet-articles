# Fact Sheet 2: 多発性関節炎 ─ 感染性vs免疫介在性の鑑別

## Sources
- Veterinary Ireland Journal: Polyarthritis differential diagnosis
- Today's Veterinary Practice: IMPA diagnosis and treatment
- MSD Veterinary Manual

## Extracted Facts

### 1. 関節液分析の比較
| 項目 | IMPA (免疫介在性) | Septic (感染性) | Reference (正常) |
|---|---|---|---|
| 有核細胞数 | >3,000 - 5,000 /μL | 著増 (often >50,000 but variable) | <3,000 /μL |
| 蛋白濃度 | >2.5 g/dL | >2.5 g/dL | <2.5 g/dL |
| 優位細胞 | **非変性好中球** (10-95%) | **変性好中球** | 単核球 |
| 細菌 | 不在 | **細胞内細菌** (確認変性好中球内) | 不在 |
| 培養 | 陰性 | 陽性 (40-50%のみ) | 陰性 |

### 2. 診断の注意点
- **感染性関節炎**:培養陽性率は40-50%程度。陰性でも感染を除外できない。
- **検体採取**: 最低3関節から採取。EDTA管（細胞計算用）とプレーン管（培養用）に分ける。
- **スメア**: 採取後直ちに作成（細胞変性を防ぐため）。

### 3. 治療プロトコル (IMPA)
- **第一選択**: プレドニゾロン (Prednisolone)
- **導入量**: **1-2 mg/kg/day PO** (大型犬は40mg/m2を使うこともある)。
- **漸減法**: 臨床的寛解後、**3-4週ごとに25%ずつ**減量。全行程2-4ヶ月以上。
- **禁忌**: NSAIDsとの併用。感染性の除外前のステロイド投与。

### 4. 治療プロトコル (Septic)
- **抗菌薬**: 感受性試験に基づく。結果判明までは広域スペクトル（Staphylococcus等をカバー）。
- **期間**: 最低 **6-8 週間**。
- **外科**: 関節洗浄 (Lavage) が推奨される。
