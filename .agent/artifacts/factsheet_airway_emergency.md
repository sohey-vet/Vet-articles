# Fact Sheet 5: 気道緊急 ─ 上気道閉塞・喉頭麻痺

## Sources
- Cornell University College of Veterinary Medicine
- Today's Veterinary Practice: Upper Airway Obstruction

## Extracted Facts

### 1. 鎮静 (Sedation) プロトコル
- 目的: 不安除去と努力性呼吸の軽減（これ自体が浮腫を軽減する）。
- **推奨薬剤**:
  - **Acepromazine**: **0.005-0.02 mg/kg IV/IM** (低用量で十分。血管拡張作用による低血圧に注意)。
  - **Butorphanol**: **0.2-0.4 mg/kg IV/IM**。
  - 併用可: Acepromazine + Butorphanol ("Neuroleptanalgesia")。

### 2. 冷却 (Active Cooling)
- 目的: 高体温（Hyperthermia）による酸素消費増大と浮腫悪化を防ぐ。
- 方法: 体表冷却（濡れタオルと送風）、冷却輸液。
- 目標体温: **39.5℃ (103°F)** 程度まで下がったら冷却中止（過冷却を防ぐ）。

### 3. ステロイド (Corticosteroids)
- 目的: 喉頭・咽頭浮腫の軽減。
- **薬剤**:
  - Dexamethasone Sodium Phosphate (Dex SP): **0.1-0.2 mg/kg IV**。
  - Prednisolone Sodium Succinate: **1-2 mg/kg IV**。
  - 単回投与が基本。

### 4. 酸素投与 (Oxygen Therapy)
- 方法:
  - Flow-by (2-3 L/min以上)。
  - Mask (密着させすぎない、嫌がるなら中止)。
  - Oxygen Cage (40-50% O2)。
  - **Nasal Catheter**: 重度の場合、片側または両側。

### 5. 緊急気道確保 (Intubation / Tracheostomy)
- 上記内科治療でSpO2 < 90% や努力呼吸が悪化する場合。
- **Propofol** (2-4 mg/kg IV to effect) または **Alfaxalone** での急速導入。
- 喉頭展開が困難な場合、緊急気管切開 (Temporary Tracheostomy) を躊躇しない。
