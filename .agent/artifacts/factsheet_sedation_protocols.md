# Fact Sheet 6: 鎮静プロトコル ─ 処置別の最適な組み合わせ (Cheat Sheet)

## Sources
- AAHA Anesthesia Guidelines
- VASG (Veterinary Anesthesia Support Group) Protocols

## Extracted Facts

### 1. 犬 (Canine) 鎮静プロトコル
#### 軽度鎮静 (Light Sedation) - 画像検査、採血など
- **Butorphanol**: 0.2-0.4 mg/kg IV/IM
  - 単独では鎮静弱い。
- **Acepromazine**: 0.01-0.03 mg/kg IV/IM
  - 鎮痛作用なし。

#### 中等度〜深い鎮静 (Procedural Sedation) - 短時間処置、創傷処置
- **Dexmedetomidine** ベース:
  - Dexmedetomidine: **3-5 mcg/kg (µg/kg) IV** または **5-10 mcg/kg IM**。
  - 併用 **Butorphanol**: 0.2 mg/kg IV/IM。
  - または **Midazolam**: 0.2 mg/kg IV/IM。
  - **拮抗薬 (Reversal)**: Atipamezole (Antisedan) - Dexmedと同じ用量(mgベースで)等量〜5倍量 IM (通常Dexmed摂取量の等量 volume で同じ концентрация なら5倍量になる)。

### 2. 猫 (Feline) 鎮静プロトコル
#### "Kitty Magic" (IM Short Anesthesia / Heavy Sedation)
- 配合例 (DKT / DMB など):
  - **Dexmedetomidine**: **10-20 mcg/kg IM**。
  - **Ketamine**: **3-5 mg/kg IM**。
  - **Butorphanol**: **0.2-0.3 mg/kg IM**。
  - 上記を混合して1ショットで筋肉内投与。
- 特徴: 15-20分の不動化が得られる。

#### 軽度鎮静・不安解除 (Pre-hospital / Hospital)
- **Gabapentin**: **50-100 mg/cat PO** (来院2-3時間前)。
- **Butorphanol**: 0.2-0.4 mg/kg IM。

### 3. Alfaxalone IM Protocol (Dog & Cat)
- **猫**:
  - Alfaxalone: **2-3 mg/kg IM** + Butorphanol 0.2 mg/kg IM。
  - 循環への影響が少なく、心疾患猫でも使いやすい（要容量多め）。
- **犬**:
  - Alfaxalone単独IMは推奨されない（興奮のリスク）。
  - 必ずMidazolamやButorphanolと併用。

### 4. モニタリング
- SpO2、心拍数、呼吸数、粘膜色。
- 純酸素投与（Flow-by）を推奨。
