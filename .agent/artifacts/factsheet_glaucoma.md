# Fact Sheet: 緑内障の緊急対応 ─ 視覚を守る72時間 (Glaucoma)

## Sources
- Veterinary Ophthalmology (Gelatt)
- RECOVER guidelines / VECCS statements for ocular emergencies

## Extracted Facts

### 1. 診断
- **眼圧（IOP）**: 正常値 15-25 mmHg (犬猫)。発作時は 40-70 mmHg 等の著しい上昇。
- 眼圧が >40-50 mmHgの状態で**48-72時間以上経過すると非可逆的な視覚喪失（網膜神経節細胞死）**を来す。
- 症状: 眼痛（目を細める・こする）、眼脂、角膜浮腫（白濁）、散瞳（PLR消失）、強膜充血・上強膜血管の怒張。

### 2. 緊急降圧プロトコル (Emergency IOP Reduction)
- **マンニトール 20% (浸透圧利尿薬)**
  - 用量: `1.0 - 2.0 g/kg (5 - 10 mL/kg) IV` over 20 - 30 分。
  - 効果発現 15-30分。最大効果で眼圧を急激に下げる。
  - 禁忌: 脱水、心不全、腎不全、前ぶどう膜炎（血液房水柵の破壊が重度の場合）。
- **Latanoprost 0.005% (ラタノプロスト / ザラカム等)**
  - プロスタグランジンアナログ。1滴を点眼。
  - **犬（特発性/原発性緑内障）において最強の降圧効果**。投与後1-2時間で眼圧を大きく下げる。1日2回まで (q12h)。
  - 禁忌: **猫（猫には無効なだけでなく炎症を悪化）**、ぶどう膜炎続発性緑内障（水晶体脱臼など炎症時）。

### 3. 維持点眼管理
- **Dorzolamide 2% (ドルゾラミド / トルソプト) + Timolol 0.5% (チモロール)**
  - 炭酸脱水酵素阻害薬 (CAI) + β遮断薬の合剤（コソプト等）。
  - 用量: 1滴を TID (q8h) または BID (q12h)。
  - 房水産生を抑制し、猫・犬問わず使用可能（ぶどう膜炎による二次性緑内障にも適用可）。

### 4. 健側の予防的点眼
- 原発性緑内障（犬の原発性閉塞隅角緑内障: PACG）の場合、片眼発症後の中央値約8ヶ月で対側眼も発症する。
- 視覚のある健眼に対して、Demecarium bromide (入手難) や βブロッカー(Timolol 1滴 q12h-24h)、または Dorzolamide 等の予防点眼を開始し、失明を先延ばしにする。
