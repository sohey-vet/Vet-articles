# Fact Sheet 16: 猫の輸液 ─ 犬との違いと注意点

## Sources
- AAHA/ISFM Feline Friendly Nursing Guidelines
- 2013 AAHA/AAFP Fluid Therapy Guidelines for Dogs and Cats

## Extracted Facts

### 1. 猫の生理学的特徴 (vs 犬)
- **血液量**: 犬 80-90 mL/kg に対し、猫は **40-60 mL/kg** と少ない。
- **過剰輸液への感受性**: 血液量が少ないため、容易に容量負荷（Volume Overload）→ **肺水腫・胸水**を起こす。
- 不感蒸泄も犬より少ないため、維持輸液要求量は低い。

### 2. 維持輸液速度 (Maintenance Rate)
- 従来の "50-60 mL/kg/day" や "132 x BW^0.75" は猫には**多すぎる**。
- **推奨維持速度**:
  - **2 - 3 mL/kg/hr**（Drip rate）。
  - 麻酔中の輸液速度: **3 mL/kg/hr**（犬は5 mL/kg/hr）。以前の10mL/kg/hrは過剰。
  - 心疾患持ちの猫: さらに減量（1-2 mL/kg/hr等）し、厳重監視。

### 3. 輸液剤の選択
- 猫はナトリウム排泄能力が低い可能性がある。
- 長期の維持輸液には、生理食塩水（0.9% NaCl）や乳酸リンゲル（LRS）などの「等張液」を使い続けるとNa負荷・量負荷になりやすい。
- **低張電解質輸液 (Hypotonic fluid)** の検討:
  - 0.45%食塩水 + 2.5%ブドウ糖 + KCl添加など（いわゆる維持液、1号・3号液的組成）。
  - ※ただしショック・脱水補正（Replacement）には必ず等張液（LRS, P-Lyteなど）を使うこと。

### 4. 過剰輸液のサイン (Fluid Overload)
- **呼吸数増加 (Tachypnea)**: 安静時呼吸数 > 30-40回/分 は危険信号。
- **呼吸努力 (Dyspnea)**: 開口呼吸は末期。
- **結膜浮腫 (Chemosis)**: 輸液過剰の早期サインとして猫で有用。
- 体重の急激な増加。

### 5. ショック輸液量 (Bolus)
- 犬: 80-90 mL/kg だが、猫は **40-60 mL/kg** が全血液量。
- ショック時のボーラス: **10 - 20 mL/kg** を15-20分かけて投与し、再評価（犬のような急速大量投与は避ける）。
